# 复古 / 玩味系

## Y2K Aesthetic
- **Keywords**: early 2000s, chrome, bubble, shiny, aqua
- **CSS Features**: chrome 渐变 / aqua buttons / 星星图标 / 亮面质感
- **Best For**: Fashion、Music、Gen-Z
- **Do Not Use For**: 企业、严肃场景
- **AI Prompt**: "Y2K aesthetic, early 2000s chrome, aqua bubble buttons, shiny metallic, nostalgic"

## Vaporwave
- **Keywords**: pastel purple-pink, retro 80s, glitch, marble
- **CSS Features**: 紫粉渐变 / 大理石贴图 / VHS 效果 / 日系汉字装饰
- **Best For**: Music、Gaming、Artist Portfolio
- **Do Not Use For**: Enterprise、Healthcare
- **AI Prompt**: "vaporwave aesthetic, pastel purple pink, retro 80s, glitch VHS, marble statue"

## Memphis Design
- **Keywords**: 80s geometric, clash pattern, bold color
- **CSS Features**: 几何装饰 / 撞色 / Z 形线 / 圆点斑马纹
- **Best For**: Creative Agency、Music、Youth Brand
- **Do Not Use For**: Finance、Medical、Legal
- **AI Prompt**: "memphis design, 80s geometric pattern, clashing colors, zigzag polka dots"

## Pixel Art
- **Keywords**: 8-bit, retro game, chunky pixel, low-res
- **CSS Features**: `image-rendering: pixelated` + 低分辨率 sprite + 8-bit 调色板
- **Best For**: Indie Game、Retro Tool、Creative Portfolio
- **Do Not Use For**: 生产力工具、专业服务

## Vintage Analog / Retro Film
- **Keywords**: film grain, sepia, warm, nostalgic
- **CSS Features**: film grain overlay / 暖色滤镜 / 怀旧 serif / 手写注释
- **Best For**: Photography、Music/Vinyl Brand
- **Do Not Use For**: Modern SaaS、Tech

## Chromatic Aberration / RGB Split
- **Keywords**: RGB offset, glitch, edge split
- **CSS Features**: `text-shadow: 2px 0 red, -2px 0 blue` / SVG filter / 鼠标动态偏移
- **Best For**: Music、Gaming、Creative
- **Do Not Use For**: 正文、信息密集

## Motion-Driven（列于此便于混搭查询）
- **Keywords**: scroll animation, reveal, parallax light
- **CSS Features**: IntersectionObserver + transform + opacity 入场
- **Best For**: 与上述任一复古风格混搭时
- **Checklist**: ☐ 尊重 `prefers-reduced-motion` ☐ 不阻塞首屏
