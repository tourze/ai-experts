# 生命周期理论

## 适用场景
- 产品生命周期管理：判断产品所处阶段，匹配对应策略。
- 客户生命周期管理：从观望到离网的全程运营。
- 与 [s-curve-growth](../../product-expert/skills/s-curve-growth/SKILL.md) 配合：S 曲线看增长动力学，生命周期看运营策略。

## 核心约束
- 产品和客户有不同的生命周期，不要混淆。
- **每个阶段的营销重点完全不同**——用成熟期的方法做导入期 = 浪费。
- 产品生命周期关键决策点：
  - 导入期：核心版本 + 教育市场 + 选择性分销。关键是找到可复制的增长模式。
  - 成长期：扩展功能 + 品牌建设 + 扩大覆盖。关键是抢市场份额。
  - 成熟期：差异化改良 + 促销/忠诚 + 全渠道。关键是在衰退前布局下一代。
  - 衰退期：精简 SKU + 降价 + 收缩渠道 + 最小投入。
- 客户生命周期从观望到离网，每阶段核心顾虑不同，运营手段必须匹配。

## 不适用场景
- 多产品组合投资决策：生命周期看单产品阶段，组合决策用 [bcg-matrix](../../product-expert/skills/bcg-matrix/SKILL.md)。
- 需要增长动力学分析：生命周期看运营策略，增长拐点和第二曲线用 [s-curve-growth](../../product-expert/skills/s-curve-growth/SKILL.md)。

## 反模式

### FAIL: 衰退期还在大量投入

```
产品月活连续 6 个月下降 -> 继续投入大量广告预算
-> 应该：减少投入，收割现金，引导客户迁移到新产品
```

### PASS: 阶段匹配的产品组合

```
产品 A（成熟期）：停止新功能投入，优化利润率，现金流输血新产品
产品 B（成长期）：追加 50% 预算，抢市场份额
产品 C（导入期）：小规模测试，找到 PMF 再加注
-> 三个产品组合覆盖不同阶段，互相输血
```
