# 产出格式

分析结果按以下四类输出：

## Quick Wins（1-2 天可落地）

```md
问题：CTA 按钮文案写"提交"
建议：改为"免费开始 — 无需信用卡"
预期：点击率提升 15-30%
难度：低
```

## High-Impact Changes（需要设计或开发）

```md
问题：首屏无社会证明
建议：在 CTA 下方添加 3 个客户 logo + "已有 2,300+ 团队使用"
预期：首屏转化提升 10-20%
难度：中
```

## Test Ideas（需要 A/B 验证）

```md
假设：长表单导致中途放弃
实验：6 字段 vs 3 字段（姓名+邮箱+公司）
主指标：提交率
副指标：线索质量、销售接通率
最小样本量：每组 500 提交
```

## Copy Alternatives（文案替代方案）

```md
原文：我们的平台功能强大且易于使用
方案 A：3 分钟搭建你的第一个自动化流程
方案 B：告别手动操作 — 省下每周 8 小时
```
