# AI 典型用语扩展清单

补充 ai-tells-checklist.md 未覆盖的 AI 写作标记。来源：Grammarly (2025)、GPTHuman (2025)、Plagiarism Today (2025)。

---

## 过度使用的动词

| 避免 | 替换为 |
|------|--------|
| bolster | strengthen, reinforce |
| underscore | emphasise, highlight, stress |
| unveil | reveal, show, present |
| enhance | improve, strengthen |
| endeavour | try, attempt |
| ascertain | find out, determine |
| elucidate | explain, clarify |
| optimise (泛用时) | improve, refine |

---

## 过度使用的形容词

| 避免 | 替换为 |
|------|--------|
| transformative | significant, major |
| cutting-edge | new, advanced, modern |
| groundbreaking | new, original |
| intricate | complex, detailed |
| nuanced | subtle, complex |
| multifaceted | complex, varied |

## 过度使用的过渡词

| 避免 | 替换为 |
|------|--------|
| furthermore | also, and |
| moreover | also, besides |
| notwithstanding | despite, still |
| that being said | however, but |
| at its core | essentially, basically |
| in the realm of | in, within |
| in the landscape of | in, within |
| in today's [任何东西] | currently, now, today |

---

## 开头/结尾套话

### 开头（直接删除或重写整句）
- "In today's fast-paced world..."
- "In the ever-evolving landscape of..."
- "Imagine a world where..."
- "Let's delve into..."

### 结尾（直接删除或重写整句）
- "By [doing X], you can [achieve Y]..."
- "In the final analysis..."
- "All things considered..."
- "At the end of the day..."

### 结构性套话
- "Whether you're a [X], [Y], or [Z]..." (三项枚举)
- "It's not just [X], it's also [Y]..."
- "Think of [X] as [详尽比喻]..."
- 以 "By" + 动名词开头：By understanding X, you can Y...

## 空洞增强词（通常可直接删除）

absolutely, basically, certainly, clearly, definitely, essentially, extremely, fundamentally, incredibly, interestingly, naturally, obviously, quite, really, significantly, simply, surely, truly, ultimately, undoubtedly, very
