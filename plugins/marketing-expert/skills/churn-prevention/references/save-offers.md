# 动态挽留 Offer 详解

## 原因 → Offer 映射

| 退出原因 | 首选 Offer | 备选 Offer |
|----------|-----------|-----------|
| 太贵了 | 20-30% 折扣（2-3 个月） | 降级到更低套餐 |
| 用得不够多 | 暂停 1-3 个月 | 使用指南 + 1v1 onboarding |
| 缺少功能 | 展示路线图 + 预计上线时间 | 加入 beta 测试 |
| 换用竞品 | 功能对比表 + 独家折扣 | 迁移回来的数据支持 |
| 技术问题 | 即时支持升级（24h 内回复） | 延长试用 + 问题修复承诺 |
| 暂时不需要 | 暂停 1-3 个月（60-80% 回归率） | 降级到免费 |
| 公司关闭 | 降级到免费计划 | 无 |

## Offer 类型参数

- **折扣**：20-30% off，持续 2-3 个月，cooldown 90 天，不叠加其他优惠。
- **暂停**：1-3 个月可选，暂停期间数据保留，60-80% 回归率。
- **降级**：保留核心功能，降低月费，保持付费关系。
- **功能解锁**：限时开放高级功能，制造体验落差。
- **人工外联**：MRR 排名前 10-20% 的客户，由 CSM 主动联系。

## Offer 展示规则

- 每次只展示一个 offer，不要给选择困难。
- 高价值客户（top 20% MRR）优先走人工外联而非自动化 offer。
- 记录 offer 接受/拒绝历史，避免对同一用户重复使用同类 offer。
