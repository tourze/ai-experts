# 参与度增强技巧

用于提升文案吸引力和转化率的结构化方法。

---

## Hook 类型表

文章开头 1-2 句必须吸引注意力。每篇选一种 hook 类型：

| Hook 类型 | 示例 | 最适合 |
|-----------|------|--------|
| 反直觉提问 | "如果'免费'方案其实每月让你亏 $500？" | 挑战假设 |
| 具体场景 | "上周二，李明打开后台发现网站已经对 Google 隐身了三周。" | 建立情感连接 |
| 惊人数据 | "73% 的 SaaS 用户在 18 个月内换平台，原因都是那三条。" | 数据驱动话题 |
| 大胆声明 | "你现在用的工具在骗你。" | 争议性话题 |
| 反常识结论 | "最便宜的选项可能是你今年最贵的决定。" | 对比评测内容 |

**规则：** 不要用通用定义开头（"X 是一种..."），不要用"在当今..."。

---

## APP 公式

Hook 之后，用 APP 公式组织引言：

- **Agree（认同）**：承认读者已有的信念或感受
- **Promise（承诺）**：告诉他们会学到什么或得到什么
- **Preview（预览）**：简要概述文章结构（长文可用迷你目录）

示例：
```
你已经知道内容营销很重要（Agree）。
这篇文章会给你一个可直接复用的季度内容计划模板（Promise）。
我们会覆盖选题、排期、分发和效果衡量四个环节（Preview）。
```

## Mini-Story 要求

每篇 2000+ 词的文章必须包含 2-3 个迷你故事（每个 50-150 词）：

**三要素：**
1. **具体人物**（用名字，即使虚构："李明""Sarah""Acme 团队"）
2. **具体场景**（日期、数字、细节）
3. **明确结果**（说明了什么观点）

**放置位置：**
- 引言或早期段落（hook 读者）
- 中段（重新吸引跳读者）
- 结尾附近（强化核心观点）

**示例（~100 词）：**
```
2024 年 3 月，Marcus 用 $5/月的最低配套餐上线了他的 SaaS。
半年后用户达到 10,000。然后他发现条款里埋着的带宽费。
$5 变成了 $89。更糟的是，迁移导致 3 周分析数据中断，
直接让他丢了一个 $2,000 的合作。
"省钱"的选择最终花了他 $3,000+。
```

## CTA 分布策略

不要只在文末放一个 CTA。分布式 CTA 转化率比仅末尾 CTA 高 121%。

| 位置 | CTA 强度 | 示例 |
|------|----------|------|
| 第一个价值段落后 | 软（了解更多） | "想看实际效果？[看看我们的功能 →]" |
| 对比/证据段落后 | 中（试试看） | "**想亲自试试？** 免费开始，无需信用卡。" |
| 文末 | 强（转化） | "**[立即开始免费试用 →]**" + 支撑文案 |

**规则：**
- CTA 必须与所在段落的内容相关（不是随机插入）
- 变换格式（行内文字、加粗 callout、按钮式）
- 第一个 CTA 应出现在前 500 词内
- 不要用 "Click here" 式的通用文案
- 每篇文章 2-3 个 CTA，不多不少

---

## 参与度检查清单

- [ ] 开头用了 hook（不是通用定义）
- [ ] 引言包含 APP 三要素
- [ ] 文中有 2-3 个带名字、细节、结果的迷你故事
- [ ] 有 2-3 个分布式 CTA（不只在结尾）
- [ ] 第一个 CTA 在前 500 词内
- [ ] 没有超过 4 句的段落
- [ ] 句长有变化（5-10 词短句 + 15-25 词长句交替）
