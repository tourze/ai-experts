# 域名脑暴（domain-name-brainstormer）

## 适用场景
- 为新产品、新栏目或新品牌寻找域名方向。
- 已有品牌名，但需要兼顾可读性、可记忆性和可注册性。
- 想把零散想法整理成一组可比较的候选列表。

## 核心约束
- 先锁定品牌语气、目标受众、行业语义和不能碰的词。
- 域名建议必须区分“主域名候选”“活动页候选”“实验性候选”。
- 没有实时注册查询时，不要把“看起来可用”说成“确定可注册”。
- 若任务转向站点层级或页面组织，切到 [site-architecture](../site-architecture/SKILL.md)。

## 代码模式
- 推荐输出模板：

```md
品牌关键词：trust / signal / growth
限制：避免生僻拼写；优先 .com / .io / .ai

| 域名 | 记忆点 | 风险 | 推荐用途 |
|---|---|---|---|
| signalforge.com | 专业、具体 | 偏 B2B | 主品牌 |
```

## 检查清单
- 是否说明了命名方向、禁用词和目标 TLD。
- 是否给出记忆点与潜在误读风险。
- 是否避免与现有品牌高度近似。
- 是否标记了需要后续注册校验的候选。

## 反模式

### FAIL: 一长串域名

```
50 个候选：
- aircloud.com
- xeniabay.io
- zenithflow.ai
- ...
"自己挑"
→ 创始人选了一个，问"这个为什么好"答不上
```

### PASS: 分组 + 理由

```md
## 主品牌候选（.com 优先）
| 域名 | 记忆点 | 风险 | 域名状态 |
| pulsewire.com | 节奏感 + 业务相关 | "wire" 偏老 | 待查 |

## 活动页候选（短 + 带 CTA）
| 域名 | 用途 | 备注 |
| trypulse.app | 试用引流 | 易记 |
```

### FAIL: 难拼写

```
推荐：xeniacore.io
→ 用户口耳相传 → 一半人拼成 "zeniacore" 或 "xenacore"
→ 直接流量损失 50%
```

### PASS: 三秒可记 + 拼写测试

```
- 听完能写出来吗？读 10 个朋友测
- 中英文都通顺吗？
- 域名拼写 ≤ 8 字符更好
- 无 "x/q/z" 易拼错字符（除非品牌就是该风格）
```
