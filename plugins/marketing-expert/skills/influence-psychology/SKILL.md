# 影响力心理学（influence-psychology）

## 适用场景
- 需要把抽象卖点转成更容易被相信和行动的表达。
- 页面、邮件或销售话术缺少可信度和推动力。
- 想系统化使用说服原则，而不是凭感觉堆社证。

## 核心约束
- 一次聚焦 1-2 个原则，避免六个原则同时上场导致过度包装。
- 所有说服要有真实证据，不允许伪造评价、权威或稀缺性。
- 先判断场景再选原则，例如首屏常用权威/社会证明，结账页常用风险逆转/稀缺。
- 若任务是文案逐句编辑，配合 [copy-editing](../copy-editing/SKILL.md)；若任务是推荐裂变机制，配合 [referral-program](../referral-program/SKILL.md)。

## 代码模式
- 推荐输出模板：

```md
目标场景：产品页首屏
首选原则：社会证明 + 权威
证据：客户 logo、案例指标、专家背书
替代表达：把“很多客户喜欢”改成“127 家付费团队在用”
```

- 各原则详解参考 [reciprocity](references/reciprocity.md)、[authority](references/authority.md)、[social-proof](references/social-proof.md)、[scarcity](references/scarcity.md)。

## 检查清单
- 是否说清楚当前页面最缺哪一种心理驱动。
- 是否为每个原则绑定了真实证据和展示位置。
- 是否避免将道德风险高的手法用在高敏感场景。
- 是否给出替代表达，而不只是理论解释。

## 反模式

### FAIL: 伪造稀缺

```
SaaS 订阅页：
⏰ 限时 24 小时！🔥 仅剩 3 个名额！💎 独家优惠！
→ SaaS 没库存，用户识别为套路，信任下降
```

### PASS: 匹配场景选原则

```
SaaS 详情页 → 社会证明（127 家付费团队）+ 权威（Forrester Leader）
结账页 → 风险逆转（30 天无理由退款）
真·稀缺场景（线下活动、名额制）→ 附真实报名列表
```

### FAIL: 6 个原则全上

```
首屏：10 万用户 + IDC 推荐 + 限时 24 小时 + 免费 7 天 + 创始人社群 + 千次演进
→ 用户启动防御，以为是诈骗
```

### PASS: 1-2 个原则 + 真实证据

```
首屏只用”社会证明 + 权威”：
- “127 家付费团队在用”（附 logo）
- “Forrester Wave 2025 Leader”（附报告链接）
→ 信息密度刚好，可验证
```
