# 4P/4C/4R 营销组合

## 适用场景
- 新产品上市前的营销策略制定。
- 现有产品营销效果不佳时的诊断。
- 与 [stp-segmentation](../stp-segmentation/SKILL.md) 配合：STP 定人群，4P 定策略。

## 核心约束
- 4P 不是四个独立决策，而是**一组协调的组合**——产品定位决定价格范围，价格影响渠道选择，渠道影响促销方式。四个 P 互相矛盾 = 营销失败。
- 4P（企业视角）-> 4C（客户视角）-> 4R（关系视角）是演进关系，不是替代。做完 4P 必须用 4C 做客户视角审视。
- 每个 P 必须有具体数字或方案。"加强推广"不是策略。
- 诊断方法：对每个 P 打分（1-5），得分最低的 = 当前营销主要短板。

## 不适用场景
- 还没确定目标客群：4P 需要先知道卖给谁，先做 [stp-segmentation](../stp-segmentation/SKILL.md)。
- 纯品牌或定位问题：4P 是营销执行层，品牌层面用 [brand-health](../brand-health/SKILL.md) 诊断。

## 检查清单
- [ ] 4P 每个维度有具体方案而非空话。
- [ ] 4P 之间相互协调（不是各做各的）。
- [ ] 用 4C 做了客户视角审视。
- [ ] 识别出了最弱的 P 并有改进计划。

## 反模式

### FAIL: 4P 各自为政

```
Product: 高端定制产品
Price: 低价促销
Place: 下沉市场电商
Promotion: 奢侈品式广告
-> 四个 P 互相矛盾，定位撕裂
```

### PASS: 4P 协调一致

```
Product: 专业级数据分析工具（面向 BI 团队）
Price: 年费 $1200/席，价值导向定价（ROI 10x）
Place: 官网直销 + Gartner/G2 评测引流
Promotion: 行业白皮书 + 在线 Demo + 客户案例
-> 四个 P 都指向"专业级 B2B SaaS"定位
```
