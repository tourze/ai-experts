# SEO 内容质量评分细则

每个维度的详细评分标准。

---

## 1. 人味/语气（权重 30%）

| 分数段 | 标准 |
|--------|------|
| 90-100 | 读起来完全像专业作者手写：有个人观点、有缩写（it's/don't）、句式自然变化、无 AI 套话 |
| 70-89 | 大部分自然，偶尔 1-2 处 AI 痕迹但不影响整体感受 |
| 50-69 | 有明显 AI 痕迹：套话（"值得注意的是""在当今环境下"）≥3 处，或段落结构过于整齐 |
| 30-49 | AI 腔重：高频使用 leverage/delve/robust 等标记词，em dash ≥5，段落全部等长 |
| 0-29 | 未经任何人性化处理的原始 AI 输出 |

**检查项：**
- AI 标记词数量（参照 content-humanizer 的 ai-tells-checklist）
- Em dash 频率
- 段落长度方差（方差过低 = 机械感）
- 是否有观点性陈述（不只是中立叙述）

---

## 2. 具体性（权重 25%）

| 分数段 | 标准 |
|--------|------|
| 90-100 | 每个核心论点都有具体数字、公司名、人名或可验证事实支撑 |
| 70-89 | 大部分论点有具体支撑，1-2 处用了"很多""一些"等模糊表述 |
| 50-69 | 混合：约一半论点有数据支撑，一半是泛泛而谈 |
| 30-49 | 主要靠抽象陈述，偶尔有数字但无来源 |
| 0-29 | 全是"研究表明""专家认为"式空洞声明 |

**检查项：**
- 文中数字/百分比/金额的数量
- 具名公司/人物/产品的数量
- "研究表明""很多公司""一般来说"等模糊表述的数量
- 数据是否标注来源和年份

---

## 3. 结构平衡（权重 20%）

| 分数段 | 标准 |
|--------|------|
| 90-100 | 散文占比 50-65%，列表/表格/代码块合理穿插，节奏感好 |
| 70-89 | 散文占比 40-70%，稍偏列表或稍偏段落但整体可读 |
| 50-69 | 散文 <40%（全是列表）或 >70%（全是大段落），阅读体验受影响 |
| 30-49 | 极端偏向一种格式，或 H2 章节长度完全一致（机械感） |
| 0-29 | 无结构或结构混乱 |

**检查项：**
- 散文占总字数的比例
- H2 章节长度方差（方差低 = 每节都 ~300 词 = AI 痕迹）
- 是否有多种格式混合（段落、列表、表格、引言）

---

## 4. SEO 合规（权重 15%）

| 分数段 | 标准 |
|--------|------|
| 90-100 | 全部达标：关键词密度 1-2%、H1 含关键词、前 100 词含关键词、2+ H2 含关键词变体、3-5 内链、2-3 外链、meta 标题 50-60 字符、meta 描述 150-160 字符 |
| 70-89 | 大部分达标，1-2 项轻微不足 |
| 50-69 | 3-4 项不达标 |
| 30-49 | 多项缺失，关键词使用有明显问题 |
| 0-29 | 基本无 SEO 意识 |

**检查项：**
- 关键词密度
- 关键词分布（H1/首段/H2/body/结尾/meta）
- 内链数量和锚文本质量
- 外链数量和权威性
- Meta 标题/描述长度和内容

---

## 5. 可读性（权重 10%）

| 分数段 | 标准 |
|--------|------|
| 90-100 | Flesch 60-70，8-10 年级，平均句长 <20 词，段落 2-4 句，有句长变化 |
| 70-89 | Flesch 50-75，基本达标，偶有长句 |
| 50-69 | Flesch 40-50 或 >80，阅读水平偏高或偏低 |
| 30-49 | 大量长句、被动语态、术语未解释 |
| 0-29 | 学术论文级难度或幼儿读物级简单 |

---

## 综合评分与路由

| 综合分 | 判定 | 动作 |
|--------|------|------|
| ≥70 | 可发布 | 进入发布流程 |
| 50-69 | 需修订 | 按优先修复项自动修订（最多 2 轮） |
| <50 | 需人工审核 | 标记问题并移交人工 |

2 轮修订后仍 <70 → 移交人工审核，附评分明细和剩余问题。
