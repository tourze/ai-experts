# 冷邮件反模式

以下做法会显著降低回复率，必须避免：

| 反模式 | 为什么有害 |
|--------|-----------|
| "I hope this finds you well" | 立刻暴露这是群发邮件，对方停止阅读。 |
| 行业术语和缩写堆砌 | 增加认知负担，显得在炫耀而非沟通。 |
| 功能列表 | 对方不关心你有什么功能，关心的是自己的问题能不能解决。 |
| HTML 排版 / 图片 / 签名 banner | 触发垃圾邮件过滤器，且看起来不像个人邮件。 |
| 假的 Re: / Fwd: 前缀 | 短期提高打开率，长期毁掉信任和域名声誉。 |
| 对所有人用同一套模板 | C-suite 和一线负责人的关注点完全不同，一套模板无法同时命中。 |
| 首封就要求 30 分钟通话 | 对陌生人来说摩擦力太高，回复率远低于"回复一个问题"。 |
| 只改名字的"个性化" | 对方一眼看穿，效果等于没有个性化。 |
| 跟进只说"checking in" | 不提供新信息的跟进是噪音，会被忽略或标记垃圾邮件。 |
