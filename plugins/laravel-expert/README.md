# laravel-expert

Laravel 框架专家插件，覆盖 Eloquent ORM、路由、队列、Livewire、安全最佳实践和 TDD 工作流。

## Skills

| Skill | 用途 |
|-------|------|
| `laravel-specialist` | Laravel 全栈开发指南（含 Eloquent/路由/队列/Livewire references） |
| `laravel-patterns` | Laravel 架构模式与控制器/服务层/API 资源 |
| `laravel-security` | 认证授权、验证、CSRF、批量赋值、文件上传安全 |
| `laravel-tdd` | PHPUnit/Pest 测试驱动开发 |
| `laravel-verification` | Laravel 项目验证工作流 |

## 安装

```bash
claude --plugin-dir /path/to/plugins/laravel-expert
```

建议同时安装 `php-expert` 插件以获得 PHP 语法检查和 PHPStan 支持。
