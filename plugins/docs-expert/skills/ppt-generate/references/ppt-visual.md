# PPT 视觉设计说明

定义演示文稿的视觉方向和设计 token，不直接生成 PPTX 文件。

## 输出内容

- 色彩方案：主色、辅色、强调色及使用比例。
- 字体系统：标题/正文/注释的字体、字号、字重层级。
- 版式规则：标题页、内容页、过渡页的网格和留白标准。
- 图表风格：数据可视化的配色和标注规范。

## 预览标准

- 每类典型页面提供 1 张 16:9 视觉参考。
- 标注关键间距、对齐线和色彩使用位置。
- 说明动画/过渡的使用原则和克制边界。

## 与 pptx 协作

- `ppt-visual` 只输出设计说明文件（`.md`）。
- `pptx` 读取设计说明并封装为可编辑的 PPTX 文件。
