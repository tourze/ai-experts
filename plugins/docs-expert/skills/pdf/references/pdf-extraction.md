# PDF 文本抽取

从 PDF 文件中提取文本和表格内容的纯抽取流程。

## 使用场景

- 只需要提取 PDF 中的文字或表格，不需要填写表单、批注或格式转换。
- 需要将 PDF 内容转成可检索的纯文本。

## 工具选择

| 工具 | 适用场景 |
|------|---------|
| `pdftotext` | 纯文本抽取，保留段落结构 |
| `camelot` / `tabula` | 表格抽取 |
| `pdfplumber` | 混合文本+表格精确抽取 |

## 约束

- 扫描件 PDF 需要先 OCR（tesseract），本流程不覆盖 OCR。
- 抽取结果默认不做语义分析，只做结构保留。
- 多列布局可能打乱阅读顺序，需要人工或后处理校正。
