# 任务清单： [FEATURE NAME]

**输入**：来自 `.specify/features/<slug>/` 的设计文档  
**前置文档**：`plan.md`（必需）、`spec.md`（必需）、`research.md`、`data-model.md`、`contracts/`

## 任务格式

`[ID] [P?] [US?] 描述`

- `[P]`：可并行执行（不同文件、无依赖）
- `[USx]`：任务归属用户故事（US1 / US2 / US3）
- 描述里必须写清楚目标文件路径
