# Citation Validator 反模式详解

## FAIL: 模糊泛称冒充引用

```
"根据行业报告..."
"多项研究表明..."
→ 无具体组织 / 日期 / URL，视为无引用
```

## PASS: 具体可追溯

```
IDC《Worldwide Semiannual AI Tracker》2024-09-15
https://www.idc.com/getdoc.jsp?containerId=xxx
→ 可 WebFetch 验证
```

## FAIL: 低质来源未降级

```
"量子计算将在 2027 年颠覆加密" [某论坛回帖, 2024]
→ E 级，但写成确定性结论
```

## PASS: 质量分级 + 降级表述

```
"有观点认为量子计算将在 2020s 后期影响现行加密体系 [社区讨论, E]，
但 NIST 后量子密码标准化进程显示主流观点更保守 [NIST 2024, A]"
```

## FAIL: 引用存在但死链

```
声明挂了 URL，但打开 404
→ 未检查就放行
```

## PASS: 验证可达性 + archive 兜底

```
WebFetch 确认 URL 可达；失败则查 archive.org 快照；
两者都失败 → 标为断链，要求换源
```

## FAIL: 数字精度可疑无源

```
"市场年增长率 37.42%"
→ 精度高但无具体来源，疑似编造
```

## PASS: 可溯源或降精度

```
要么补原始来源（含报告名 + 页码），
要么改为"约 37%"并注明估算方法
```
