# Citation Validator 输出模板

## 报告骨架

```markdown
# Citation Validation Report

## Executive Summary
- 总声明数 / 带引用数 / 完整引用数 / 核实支持数 / 可疑幻觉数
- 来源质量分布：A x / B x / C x / D x / E x
- 总体质量分：x/10

## Critical Issues（需立即整改）
- [声明片段] — [问题：无引用 / 引用死链 / 引用不支持 / 源质量过低]

## Detailed Findings
[按章节或声明逐条分析]

## Recommendations（按优先级）
1. ...
```

## 质量分计算

| 分数 | 含义 |
|------|------|
| 9-10 | 专业研究级 |
| 7-8 | 多数场合可接受 |
| 5-6 | 有问题需改 |
| 3-4 | 问题严重 |
| 0-2 | 不可信 |

## 成功标准

- 100% 事实声明有引用
- 100% 引用四要素齐全
- 95%+ 引用支持声明
- 无未解释的幻觉
- 平均源质量 ≥ B
- 总体分 ≥ 8/10

## 高风险场景特殊要求

### 医疗 / 健康

- 要求 A-B 级（同行评审 / 临床指南）
- 验证 PubMed PMID
- 区分"已证实"与"初步"

### 法律 / 合规

- 引用原始法规文件
- 标明 docket 编号
- 标明司法管辖范围

### 市场 / 财务

- 优先 SEC 文件、公司年报、原始披露
- 注明报告期（YYYY-Q）
- 区分 GAAP / non-GAAP
