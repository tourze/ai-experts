# Services, Hospitality & Creative Presets

## 27. Beauty / Spa / Wellness
- Style: Soft UI Evolution + Elegant
- Colors: `#E8B4B8` / `#A8D5BA` / 金 `#D4AF37` / bg `#FFF5F5`
- Fonts: Cormorant Garamond / Montserrat
- Effects: 柔阴影、缓慢过渡、大图
- AVOID: 霓虹、锐利动效、暗黑默认

## 28. Restaurant
- Style: Editorial Grid + Large Imagery
- Colors: `#18181B` / `#FBBF24` / bg `#FAFAF9`
- Fonts: Playfair Display / Lato
- Effects: 菜品大图、预订 CTA 突出
- AVOID: 密集文字、卡通

## 29. Hotel / Travel
- Style: Parallax Storytelling + Large Imagery
- Colors: `#0C4A6E` / `#F59E0B` / bg `#FAFAFA`
- Fonts: Newsreader / Inter
- Effects: 滚动叙事、地图、沉浸图
- AVOID: 堆信息、弱视觉

## 30. Legal / Law Firm
- Style: Swiss Modernism 2.0 + Serif
- Colors: `#0F172A` / `#B08D57` / bg `#F8FAFC`
- Fonts: Libre Caslon Text / Lato
- Effects: 克制、衬线、留白
- AVOID: 活泼、亮色、Emoji

## 31. Home Services / Local Business
- Style: Flat + Trust Signals
- Colors: `#0284C7` / `#F59E0B` / bg `#FFFFFF`
- Fonts: Inter / Inter
- Effects: 评分/证件/团队照突出
- AVOID: 过精致、过简化到无信息

## 32. Design Portfolio / Agency
- Style: Brutalism + Interactive Cursor
- Colors: `#000000` / `#FAFAFA` / 偶尔亮 accent
- Fonts: Syne / Inter
- Effects: 大图 1:1、自定义光标、滚动揭示
- AVOID: 模板感 SaaS 风、温和圆角

## 33. Photography
- Style: Exaggerated Minimalism + Full-bleed
- Colors: `#0A0A0A` / bg `#FFFFFF` 或 `#0A0A0A`（双模式）
- Fonts: Tenor Sans / Lora
- Effects: 全屏图、边缘裁切、图网格
- AVOID: 装饰干扰图、多配色

## 34. Gaming (Web/Product)
