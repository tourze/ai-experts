# Local Retail & Services Presets

## 56. Florist
- Style: Editorial + Nature Distilled
- Colors: `#16A34A` / `#DB2777` / bg `#FEFCE8`
- Fonts: Cormorant Garamond / Montserrat
- Effects: 花束特写、季节系列、订购流程清晰
- AVOID: 企业蓝、高科技感

## 57. Bakery / Patisserie
- Style: Claymorphism + Warm
- Colors: `#B45309` / `#F59E0B` / bg `#FFFBEB`
- Fonts: Caprasimo / DM Sans
- Effects: 食物微距、暖色、手写字注释
- AVOID: 冷色极简、工业感

## 58. Brewery / Craft Beer
- Style: Brutalism + Vintage
- Colors: `#78350F` / `#FBBF24` / bg `#0A0A0A`
- Fonts: Alfa Slab One / Inter
- Effects: 酒标设计感、暗色背景、粗糙肌理
- AVOID: 柔和甜美、企业 SaaS 风

## 59. Cafe / Coffee Shop
- Style: Soft UI + Editorial
- Colors: `#78350F` / `#10B981` / bg `#FFFBEB`
- Fonts: Fraunces / Inter
- Effects: 温暖拍摄、菜单编辑化、营业时间突出
- AVOID: 快节奏动效、霓虹

## 60. Barber / Hair Salon (Male-oriented)

- Fonts: Archivo Black / Inter
- Effects: 黑白人像、粗体 label、简洁预约
