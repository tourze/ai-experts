# Tech / SaaS Presets

## 1. SaaS (General)
- Style: Flat + 轻 Glassmorphism
- Colors: `#2563EB` / `#EA580C` / bg `#F8FAFC`
- Fonts: Inter / Inter
- Effects: Subtle hover 200ms、浅阴影卡片
- AVOID: 过度动画、默认暗黑、AI 紫粉渐变

## 2. Developer Tool / IDE / CLI
- Style: Minimalism + Mono Accent
- Colors: `#0F172A` / `#22D3EE` / bg `#020617`（dark 默认）
- Fonts: Geist / Geist Mono
- Effects: 等宽数字、代码块、无装饰
- AVOID: 儿童感、AI 渐变、图标过载

## 3. AI / Chatbot / Copilot
- Style: AI-Native + Soft Gradient
- Colors: `#0F172A` / `#8B5CF6`（克制） / bg 渐变 `#020617→#0F172A`
- Fonts: Geist / Inter
- Effects: 流式生成动效、粒子/光晕
- AVOID: 纯紫粉渐变一条道、AI=魔法光

## 4. Cybersecurity
- Style: HUD + Swiss Modernism 2.0
- Colors: `#0F172A` / `#10B981` / 警 `#EF4444` / bg `#020617`
- Fonts: IBM Plex Sans / IBM Plex Mono
- Effects: 等宽数据、网格、技术图示
- AVOID: 卡通、渐变、过度动画

## 5. Micro SaaS / Indie
- Style: Minimal & Direct + Vibrant Accent
- Colors: `#6366F1` / `#059669` / bg `#F5F3FF`
- Fonts: Plus Jakarta Sans / Inter
- Effects: 演示 GIF/视频置顶、极速滚动
- AVOID: 企业级堆特性、长篇大论

## 6. DevTools Observability / CI
- Style: Minimalism + Mono Code
- Colors: `#020617` / `#10B981` / 错 `#EF4444` / bg `#0F172A`
- Fonts: Geist / Geist Mono
- Effects: 等宽日志、彩色代码块
- AVOID: 卡通、光鲜

## 7. Enterprise B2B Platform
- Style: Dimensional Layering + Data-Dense
- Colors: `#0F172A` / `#3B82F6` / bg `#F8FAFC`
- Fonts: IBM Plex Sans / IBM Plex Sans
- Effects: 层级阴影、数据密集卡片
- AVOID: 消费级活泼、AI 紫粉

## 8. Data / BI / Analytics
- Style: Data-Dense Dashboard + Small Multiples
- Colors: `#1E293B` / ColorBrewer 色盲安全板 / bg `#F8FAFC`
- Fonts: Inter / JetBrains Mono
- Effects: Small multiples、精细 tooltip
- AVOID: 3D 图、双 Y 轴（见 [data-visualization](../../../../data-ai-expert/skills/data-visualization/SKILL.md)）
