# 有机 / 自然系

## Organic Biophilic
- **Keywords**: nature-inspired, leaf, water, earthy palette
- **CSS Features**: 有机曲线 (`border-radius: 40% 60% 30% 70%`) / 自然色（绿/木/土） / 大留白 / 手绘插画
- **Best For**: Wellness、Sustainability、Health、Spa
- **Do Not Use For**: Tech、B2B SaaS、Data-dense
- **AI Prompt**: "organic biophilic design, nature-inspired curves, earthy green brown palette, leaf water motifs"
- **Checklist**: ☐ 不规则圆角 ☐ 自然色 ☐ 曲线元素 ☐ 留白充足

## Nature Distilled
- **Keywords**: minimal natural, zen, sparse, quiet
- **CSS Features**: 极少色彩（1-2 色自然色） + 大量留白 + 纤细衬线字
- **Best For**: Luxury Wellness、Tea/Coffee Brand、Quiet Product
- **Do Not Use For**: 活跃 App、需要强调感

## Biomimetic / Organic 2.0
- **Keywords**: biological pattern, living UI, breathing
- **CSS Features**: 呼吸动画 / 有机 morph / 生物启发曲线
- **Best For**: Biotech、Sustainable Tech、Meditation
- **Do Not Use For**: Dashboard、企业
- **AI Prompt**: "biomimetic organic ui, living breathing elements, biological patterns, morphing curves"

## Dark Mode (OLED)
- **Keywords**: true black, low emission, code editor
- **CSS Features**: bg `#000000` 或 `#0A0A0A` / 文本 `#E5E5E5` / 克制 accent
- **Best For**: Developer Tool、Night-mode App、Coding Platform
- **Do Not Use For**: 默认模式的白天型产品、老年用户
- **AI Prompt**: "dark mode OLED, true black background, low emission, minimalist night theme"
- **Checklist**: ☐ bg ≤ #0A0A0A ☐ 文本对比 > 4.5:1 ☐ 色彩饱和度降低 ☐ 阴影用高光替代

## E-Ink / Paper
- **Keywords**: paper texture, high contrast black-white, book-like
- **CSS Features**: 纸张肌理 bg / 纯黑文本 / Serif / 最小交互装饰
- **Best For**: Reading App、Digital Newspaper、Kindle-like
- **Do Not Use For**: 富媒体、视频、App Store

## Accessible & Ethical
- **Keywords**: high contrast, WCAG AAA, simple, inclusive
- **CSS Features**: 对比 ≥ 7:1 / 字号 ≥ 16px / 无装饰色 / 明确 focus ring
- **Best For**: Government、Healthcare、Education、Public Service
- **Do Not Use For**: 品牌营销页（会显得过于朴素）
- **Checklist**: ☐ WCAG AAA ☐ 无颜色单独承载信息 ☐ 键盘完整可达 ☐ 减少动效
