# 有机 / 自然系

## Organic Biophilic（有机亲生物）
- **关键词**: 自然灵感、叶、水、大地色系
- **CSS 特征**: 有机曲线 (`border-radius: 40% 60% 30% 70%`) / 自然色（绿/木/土） / 大留白 / 手绘插画
- **最适合**: 健康养生、可持续发展、保健、水疗
- **不适合**: 科技、B2B SaaS、数据密集型
- **AI Prompt**: "organic biophilic design, nature-inspired curves, earthy green brown palette, leaf water motifs"
- **检查清单**: ☐ 不规则圆角 ☐ 自然色 ☐ 曲线元素 ☐ 留白充足

## Nature Distilled（自然萃取）
- **关键词**: 极简自然、禅意、稀疏、静谧
- **CSS 特征**: 极少色彩（1-2 色自然色） + 大量留白 + 纤细衬线字
- **最适合**: 高端养生、茶/咖啡品牌、静谧产品
- **不适合**: 活跃 App、需要强调感

## Biomimetic / Organic 2.0（仿生/有机 2.0）
- **关键词**: 生物纹理、活体界面、呼吸感
- **CSS 特征**: 呼吸动画 / 有机变形 / 生物启发曲线
- **最适合**: 生物科技、可持续科技、冥想
- **不适合**: 仪表盘、企业
- **AI Prompt**: "biomimetic organic ui, living breathing elements, biological patterns, morphing curves"

## Dark Mode (OLED)（深色模式 OLED）
- **关键词**: 纯黑、低功耗、代码编辑器
- **CSS 特征**: bg `#000000` 或 `#0A0A0A` / 文本 `#E5E5E5` / 克制 accent
- **最适合**: 开发者工具、夜间模式 App、编码平台
- **不适合**: 默认白天型产品、老年用户
- **AI Prompt**: "dark mode OLED, true black background, low emission, minimalist night theme"
- **检查清单**: ☐ bg ≤ #0A0A0A ☐ 文本对比 > 4.5:1 ☐ 色彩饱和度降低 ☐ 阴影用高光替代

## E-Ink / Paper（电子墨水/纸张）
- **关键词**: 纸张纹理、黑白高对比、书本感
- **CSS 特征**: 纸张肌理 bg / 纯黑文本 / Serif / 最小交互装饰
- **最适合**: 阅读 App、数字报纸、Kindle 风格
- **不适合**: 富媒体、视频、应用商店

## Accessible & Ethical（可访问与道德设计）
- **关键词**: 高对比、WCAG AAA、简洁、包容
- **CSS 特征**: 对比 ≥ 7:1 / 字号 ≥ 16px / 无装饰色 / 明确 focus ring
- **最适合**: 政府、医疗、教育、公共服务
- **不适合**: 品牌营销页（会显得过于朴素）
- **检查清单**: ☐ WCAG AAA ☐ 无颜色单独承载信息 ☐ 键盘完整可达 ☐ 减少动效
