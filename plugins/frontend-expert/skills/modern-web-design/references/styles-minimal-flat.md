# 极简 / 扁平系

## Minimalism / Swiss Style
- **Keywords**: clean, sans-serif, grid-based, white space, high contrast
- **CSS Features**: 12-16 列栅格、`box-shadow: none`、`border-radius: 0`、单 accent 色、`font-weight: 400-700`、大量 `padding`
- **Best For**: Enterprise、Dashboard、Documentation、B2B SaaS
- **Do Not Use For**: 创意组合、娱乐、Gen-Z
- **AI Prompt**: "minimalist swiss design, white space, geometric grid, sans-serif, high contrast black and white, clean typography"
- **Checklist**: ☐ 栅格对齐 ☐ 无装饰阴影 ☐ WCAG AAA 对比 ☐ 单 accent ☐ 大留白

## Flat Design
- **Keywords**: 2D, no shadow, solid color, simple icons
- **CSS Features**: 纯色块 / 无阴影 / `border-radius: 4-8px` / 扁平图标 / 单层 UI
- **Best For**: Web App、Mobile App、Startup MVP
- **Do Not Use For**: 需要深度感的奢侈品、真实感模拟
- **AI Prompt**: "flat design, solid colors, no gradients, simple geometric shapes, 2D illustration"
- **Checklist**: ☐ 无阴影 ☐ 色块分明 ☐ 图标扁平 ☐ 无高光/内阴影

## Exaggerated Minimalism
- **Keywords**: oversized type, extreme whitespace, sparse, editorial
- **CSS Features**: `font-size: clamp(4rem, 10vw, 12rem)` display / 极大 `padding-y` / 极少内容
- **Best For**: Fashion、Architecture、Portfolio、Luxury
- **Do Not Use For**: 信息密集、工具型产品
- **AI Prompt**: "oversized typography, massive whitespace, editorial minimalism, sparse layout"
- **Checklist**: ☐ Display 字号 > 8rem ☐ 视口 50% 以上留白 ☐ 无冗余装饰

## Swiss Modernism 2.0
- **Keywords**: modern grid, Akzidenz, corporate, editorial structure
- **CSS Features**: 严格 12 列 / Helvetica/Inter/Akzidenz / 无装饰 / 大量对齐线
- **Best For**: Corporate、Architecture Firm、Editorial
- **Do Not Use For**: Playful、小微业务
- **AI Prompt**: "swiss modernism, strict grid system, helvetica, editorial corporate, precise alignment"

## Soft UI Evolution
- **Keywords**: subtle depth, pastel, rounded, enterprise-modern
- **CSS Features**: `border-radius: 12-16px` / 极轻阴影 `0 1px 3px rgba(0,0,0,0.04)` / 柔和色
- **Best For**: Modern Enterprise SaaS、Fintech、Professional Tools
- **Do Not Use For**: 需要强品牌冲击、Brutalist 气质
- **AI Prompt**: "soft ui, pastel palette, rounded corners 16px, subtle shadows, modern enterprise"
- **Checklist**: ☐ 圆角统一 12-16px ☐ 阴影深度 ≤ 4px ☐ 色彩饱和度偏低
