# 极简 / 扁平系

## Minimalism / Swiss Style（极简主义/瑞士风格）
- **关键词**: 干净、无衬线、栅格化、留白、高对比
- **CSS 特征**: 12-16 列栅格、`box-shadow: none`、`border-radius: 0`、单 accent 色、`font-weight: 400-700`、大量 `padding`
- **最适合**: 企业、仪表盘、文档、B2B SaaS
- **不适合**: 创意作品集、娱乐、Z 世代
- **AI Prompt**: "minimalist swiss design, white space, geometric grid, sans-serif, high contrast black and white, clean typography"
- **检查清单**: ☐ 栅格对齐 ☐ 无装饰阴影 ☐ WCAG AAA 对比 ☐ 单 accent ☐ 大留白

## Flat Design（扁平设计）
- **关键词**: 2D、无阴影、纯色、简约图标
- **CSS 特征**: 纯色块 / 无阴影 / `border-radius: 4-8px` / 扁平图标 / 单层 UI
- **最适合**: Web App、移动 App、初创 MVP
- **不适合**: 需要深度感的奢侈品、真实感模拟
- **AI Prompt**: "flat design, solid colors, no gradients, simple geometric shapes, 2D illustration"
- **检查清单**: ☐ 无阴影 ☐ 色块分明 ☐ 图标扁平 ☐ 无高光/内阴影

## Exaggerated Minimalism（夸张极简）
- **关键词**: 超大字体、极限留白、稀疏、编辑风
- **CSS 特征**: `font-size: clamp(4rem, 10vw, 12rem)` display / 极大 `padding-y` / 极少内容
- **最适合**: 时尚、建筑、作品集、奢侈品
- **不适合**: 信息密集型、工具型产品
- **AI Prompt**: "oversized typography, massive whitespace, editorial minimalism, sparse layout"
- **检查清单**: ☐ Display 字号 > 8rem ☐ 视口 50% 以上留白 ☐ 无冗余装饰

## Swiss Modernism 2.0（瑞士现代主义 2.0）
- **关键词**: 现代网格、Akzidenz、企业、编辑结构
- **CSS 特征**: 严格 12 列 / Helvetica/Inter/Akzidenz / 无装饰 / 大量对齐线
- **最适合**: 企业、建筑事务所、编辑出版
- **不适合**: 趣味型、小微业务
- **AI Prompt**: "swiss modernism, strict grid system, helvetica, editorial corporate, precise alignment"

## Soft UI Evolution（柔和界面进化）
- **关键词**: 微妙深度、柔和色、圆角、企业现代
- **CSS 特征**: `border-radius: 12-16px` / 极轻阴影 `0 1px 3px rgba(0,0,0,0.04)` / 柔和色
- **最适合**: 现代企业 SaaS、金融科技、专业工具
- **不适合**: 需要强品牌冲击、粗野主义气质
- **AI Prompt**: "soft ui, pastel palette, rounded corners 16px, subtle shadows, modern enterprise"
- **检查清单**: ☐ 圆角统一 12-16px ☐ 阴影深度 ≤ 4px ☐ 色彩饱和度偏低
