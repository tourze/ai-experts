# 文化 / 历史系

## Bauhaus（包豪斯）
- **关键词**: 几何原色、栅格、工业、功能主义
- **CSS 特征**: 基本几何（圆/方/三角）+ 三原色 `#E7322F` + `#FFD500` + `#0060AA` + 黑白 + 栅格
- **最适合**: 建筑、工业设计、艺术项目、设计教育
- **不适合**: 消费 App、Z 世代
- **AI Prompt**: "Bauhaus design, geometric primary colors, red yellow blue, functional industrial, grid system"
- **检查清单**: ☐ 基本几何形状 ☐ 三原色纯度 ☐ 栅格对齐 ☐ 功能主义

## Art Deco（装饰艺术）
- **关键词**: 对称、金色、棱角、1920年代奢华
- **CSS 特征**: 对称装饰 / 金线 `#D4AF37` / 几何 frames / 深色背景
- **最适合**: 奢华酒店、盖茨比美学、精品品牌
- **不适合**: 科技 SaaS、开发者工具
- **AI Prompt**: "Art Deco design, symmetrical geometric gold ornaments, 1920s luxury, deep backgrounds"

## Art Nouveau（新艺术运动）
- **关键词**: 流动曲线、植物、花卉、有机线条
- **CSS 特征**: SVG 曲线装饰 / 花卉纹样 / 手绘感 / 金褐色
- **最适合**: 精品店、健康养生、手工艺品牌
- **不适合**: 企业、仪表盘
- **AI Prompt**: "Art Nouveau style, flowing botanical curves, floral ornaments, organic lines, golden brown"

## Pop Art（波普艺术）
- **关键词**: 网点、漫画、大胆平色、沃霍尔
- **CSS 特征**: 网点 pattern / 大色块 / 粗黑线 / 对话气泡
- **最适合**: 青年品牌、音乐、流行文化产品
- **不适合**: B2B、医疗、法律
- **AI Prompt**: "Pop Art style, halftone dots, bold flat colors, comic book, Warhol aesthetic"

## Nordic / Scandinavian（北欧/斯堪的纳维亚）
- **关键词**: 舒适温馨、温暖极简、木色、柔和、惬意
- **CSS 特征**: 暖白 `#FAFAF7` / 木色 / 浅灰蓝 / 细线 / 大圆角 12-16px
- **最适合**: 家居用品、家具、健康养生、生活方式品牌
- **不适合**: 游戏、高科技、紧迫感产品
- **AI Prompt**: "Nordic scandinavian design, hygge warm minimal, wood pastel palette, cozy aesthetic"
- **检查清单**: ☐ 暖中性色 ☐ 留白足 ☐ 柔和圆角 ☐ 无炫色

## Japanese Ma / Wabi-sabi（日本间/侘寂）
- **关键词**: 留白、非对称、不完美之美、静谧
- **CSS 特征**: 极大留白 / 非对称构图 / 自然材质感 / 单一强调色
- **最适合**: 茶品牌、传统工艺、日式高端
- **不适合**: 高密度工具、快节奏 App
- **AI Prompt**: "Japanese ma wabi-sabi, negative space, asymmetric, imperfect natural beauty, minimalist"

## Cottagecore（田园风）
- **关键词**: 乡村浪漫、花卉、柔和色、怀旧
- **CSS 特征**: 花卉 border / 柔和米色 / Serif 手写混排 / 纸张纹理
- **最适合**: 生活博客、手工商店、小众养生
- **不适合**: 科技、B2B
- **AI Prompt**: "cottagecore aesthetic, rural romance florals, soft pastel vintage, handwritten serif"

## Dark Academia（暗黑学院风）
- **关键词**: 学术、沉郁、棕色系、图书馆、哥特
- **CSS 特征**: 深棕 `#3C2A1E` / 米黄 / Serif / 书本纹理 / 低光照
- **最适合**: 教育、书籍、文学品牌
- **不适合**: 年轻消费品、科技现代
- **AI Prompt**: "dark academia, scholarly library mood, brown beige palette, gothic serif, candlelight"

## Folk / Handmade（民艺/手作）
- **关键词**: 手绘、不完美、工艺、纺织纹样
- **CSS 特征**: 手绘 SVG 装饰 / 民间花纹 / 土色 / 不规则边
- **最适合**: 手工艺、公平贸易、文化项目
- **不适合**: SaaS、企业
- **AI Prompt**: "folk handmade craft, hand-drawn imperfect, textile pattern, earthen tones"

## Steampunk（蒸汽朋克）
- **关键词**: 维多利亚机械、黄铜、齿轮、铜色
- **CSS 特征**: 齿轮装饰 / 铜色 `#B87333` / 深棕 / 维多利亚装饰边
- **最适合**: 小众游戏、叙事项目、主题活动
- **不适合**: 现代 SaaS、移动 App
- **AI Prompt**: "steampunk aesthetic, Victorian brass gears, copper machinery, industrial ornaments"
