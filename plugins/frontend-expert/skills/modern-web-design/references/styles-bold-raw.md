# 粗砺 / 大声量系

## Brutalism
- **Keywords**: raw, harsh, monospace, no rounded corners, utilitarian
- **CSS Features**: `border-radius: 0` / `border: 2px solid #000` / 字体对比 / 裸 HTML 感 / 单色或反差色
- **Best For**: Design Portfolio、Art Project、Statement Brand
- **Do Not Use For**: Banking、Healthcare、Consumer App
- **AI Prompt**: "brutalism web design, raw harsh typography, no rounded corners, monochrome, utilitarian aesthetic"
- **Checklist**: ☐ 无圆角 ☐ 粗边框 ☐ 等宽字或 Display 粗 ☐ 刻意"不漂亮"

## Neubrutalism
- **Keywords**: bold shadow, solid color, thick border, Figma-style
- **CSS Features**: `border: 3px solid #000` + `box-shadow: 4px 4px 0 #000`（offset solid 阴影） + 饱和色 + 大圆角对比粗边
- **Best For**: Gen-Z、Startup、Figma-era Products
- **Do Not Use For**: Legal、Corporate、Accessibility-critical
- **AI Prompt**: "neubrutalism, bold offset shadows, thick black borders, bright solid colors, playful rebellious"
- **Checklist**: ☐ 边框 ≥ 2px solid ☐ offset 阴影（非 blur） ☐ 饱和色 ≥ 2 个

## Anti-Polish / Raw Aesthetic
- **Keywords**: imperfect, handmade, sketch, indie
- **CSS Features**: 手写字体 / 歪斜元素 / 低保真装饰 / 反设计感
- **Best For**: Creative Portfolio、Artist Site、Indie Creator
- **Do Not Use For**: 企业、工具类、信任密集型
- **AI Prompt**: "anti-polish raw aesthetic, imperfect handmade sketch, indie creative, low-fi decoration"

## Gen-Z Chaos / Maximalism
- **Keywords**: sticker, clash, layered, vibrant, dense
- **CSS Features**: 多字体 / 多色 / 贴纸元素 / 歪斜 `transform: rotate(-3deg)` / 重叠
- **Best For**: Gen-Z Lifestyle、Music Artist、Youth Brand
- **Do Not Use For**: B2B、Enterprise、严肃场景
- **AI Prompt**: "gen-z maximalism, sticker collage, clashing colors, layered chaos, vibrant dense composition"

## Kinetic Typography
- **Keywords**: moving text, animated type, scroll-driven, oversized
- **CSS Features**: 大字号 / scroll-triggered 动画 / text-mask / variable font 动态
- **Best For**: Hero Section、Marketing Site、Product Launch
- **Do Not Use For**: 正文、信息密集
- **AI Prompt**: "kinetic typography, animated oversized text, scroll-driven motion, variable font"

## Editorial Grid / Magazine
- **Keywords**: multi-column, serif headline, pull quote, asymmetric
- **CSS Features**: 非对称栅格 / 大标题衬线 / 多栏正文 / 图文混排
- **Best For**: News、Blog、Magazine、Long-form
- **Do Not Use For**: App、Dashboard、工具
- **AI Prompt**: "editorial magazine layout, multi-column grid, serif headlines, pull quotes, asymmetric"
- **Checklist**: ☐ 多栏 ☐ Display Serif ☐ 大标题 ☐ 非对称结构
