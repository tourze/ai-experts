# 未来主义 / 技术系

## Cyberpunk UI
- **Keywords**: neon, glitch, dark, cyan-magenta, dystopian
- **CSS Features**: bg `#020617` + neon `#22D3EE`/`#EC4899` + glitch animation + 扫描线 + 发光
- **Best For**: Gaming、Web3、Crypto、Tech Product
- **Do Not Use For**: Banking、Healthcare、Legal
- **AI Prompt**: "cyberpunk ui, neon cyan magenta, dark dystopian, glitch effects, scanlines, glowing text"
- **Checklist**: ☐ 深色背景 ☐ 饱和霓虹色 ☐ 发光 text-shadow ☐ 可选 glitch 动画

## HUD / Sci-Fi FUI
- **Keywords**: heads-up display, futuristic UI, grid, data stream
- **CSS Features**: 等宽字 + 线框图 + 数据流动画 + 克制霓虹 + 网格
- **Best For**: Sci-fi Game、Space Tech、Cybersecurity Platform
- **Do Not Use For**: Consumer App、Mobile
- **AI Prompt**: "HUD sci-fi FUI, futuristic heads-up display, wireframe grids, data stream animations"

## Retro-Futurism
- **Keywords**: 80s future, chrome, grid horizon, synthwave
- **CSS Features**: 紫粉渐变 + 网格地平线 + chrome text + 80s palette
- **Best For**: Gaming、Entertainment、Music
- **Do Not Use For**: Fintech、Medical
- **AI Prompt**: "retro-futurism, 80s synthwave, chrome text, grid horizon, purple-pink gradient"

## Aurora UI
- **Keywords**: flowing gradient, soft light, ambient
- **CSS Features**: 多色 radial-gradient 叠加 + 动态变换 + 半透明 overlay
- **Best For**: Modern SaaS、Creative Agency、AI Product
- **Do Not Use For**: 密集仪表盘、严肃 B2B
- **AI Prompt**: "aurora ui, flowing soft light gradients, ambient atmospheric, multi-color blend"
- **Checklist**: ☐ ≥ 2 个 radial-gradient ☐ 低饱和 ☐ 不喧宾夺主

## Gradient Mesh / Aurora Evolved
- **Keywords**: complex gradient, blurred mesh, generative
- **CSS Features**: SVG/Canvas 生成 mesh + blur + 多点色彩
- **Best For**: Hero Background、Creative
- **Do Not Use For**: 正文区域、信息面板

## AI-Native UI
- **Keywords**: generative, streaming, chat, copilot, soft glow
- **CSS Features**: 流式生成动画（逐字打字） + 低饱和渐变 + 粒子/光晕 + Mono 字体混用
- **Best For**: AI Chatbot、Copilot、LLM Platform
- **Do Not Use For**: 非 AI 产品（套 AI 风会显得蹭）
- **AI Prompt**: "AI-native ui, generative streaming interface, soft glow, copilot chat, subtle gradient"
- **AVOID**: 纯紫粉渐变一条道、把 AI = 魔法光

## Spatial UI (VisionOS)
- **Keywords**: 3D space, depth, glassmorphism, eye-tracked
- **CSS Features**: Glass + 3D transform + 景深模糊 + 大圆角
- **Best For**: Spatial Computing、VR/AR、Apple Vision Pro
- **Do Not Use For**: 2D Web、Mobile
