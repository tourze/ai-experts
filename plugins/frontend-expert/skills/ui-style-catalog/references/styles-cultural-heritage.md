# 文化 / 历史系

## Bauhaus
- **Keywords**: geometric primary, grid, industrial, functional
- **CSS Features**: 基本几何（圆/方/三角）+ 三原色 `#E7322F` + `#FFD500` + `#0060AA` + 黑白 + 栅格
- **Best For**: 建筑、工业设计、艺术项目、设计教育
- **Do Not Use For**: 消费 App、Gen-Z
- **AI Prompt**: "Bauhaus design, geometric primary colors, red yellow blue, functional industrial, grid system"
- **Checklist**: ☐ 基本几何形状 ☐ 三原色纯度 ☐ 栅格对齐 ☐ 功能主义

## Art Deco
- **Keywords**: symmetry, gold, angular, 1920s luxury
- **CSS Features**: 对称装饰 / 金线 `#D4AF37` / 几何 frames / 深色背景
- **Best For**: Luxury Hotel、Gatsby Aesthetic、Boutique Brand
- **Do Not Use For**: Tech SaaS、Developer Tool
- **AI Prompt**: "Art Deco design, symmetrical geometric gold ornaments, 1920s luxury, deep backgrounds"

## Art Nouveau
- **Keywords**: flowing curves, botanical, floral, organic lines
- **CSS Features**: SVG 曲线装饰 / 花卉纹样 / 手绘感 / 金褐色
- **Best For**: Boutique、Wellness、Artisan Brand
- **Do Not Use For**: Enterprise、Dashboard
- **AI Prompt**: "Art Nouveau style, flowing botanical curves, floral ornaments, organic lines, golden brown"

## Pop Art
- **Keywords**: halftone, comic, bold flat color, Warhol
- **CSS Features**: 网点 pattern / 大色块 / 粗黑线 / 对话气泡
- **Best For**: Youth Brand、Music、Pop Culture Product
- **Do Not Use For**: B2B、Healthcare、Legal
- **AI Prompt**: "Pop Art style, halftone dots, bold flat colors, comic book, Warhol aesthetic"

## Nordic / Scandinavian
- **Keywords**: hygge, warm minimal, wood, pastel, cozy
- **CSS Features**: 暖白 `#FAFAF7` / 木色 / 浅灰蓝 / 细线 / 大圆角 12-16px
- **Best For**: Home Goods、Furniture、Wellness、Lifestyle Brand
- **Do Not Use For**: Gaming、High-tech、Urgent Product
- **AI Prompt**: "Nordic scandinavian design, hygge warm minimal, wood pastel palette, cozy aesthetic"
- **Checklist**: ☐ 暖中性色 ☐ 留白足 ☐ 柔和圆角 ☐ 无炫色

## Japanese Ma / Wabi-sabi
- **Keywords**: negative space, asymmetry, imperfect beauty, quiet
- **CSS Features**: 极大留白 / 非对称构图 / 自然材质感 / 单一强调色
- **Best For**: Tea Brand、Traditional Craft、Luxury Japanese
- **Do Not Use For**: 高密度工具、快节奏 App
- **AI Prompt**: "Japanese ma wabi-sabi, negative space, asymmetric, imperfect natural beauty, minimalist"

## Cottagecore
- **Keywords**: rural romance, florals, soft pastel, vintage

- **Best For**: Lifestyle Blog、Craft Shop、Wellness Niche
- **Do Not Use For**: Tech、B2B
- **AI Prompt**: "cottagecore aesthetic, rural romance florals, soft pastel vintage, handwritten serif"

## Folk / Handmade
- **Keywords**: hand-drawn, imperfect, craft, textile pattern
- **CSS Features**: 手绘 SVG 装饰 / 民间花纹 / 土色 / 不规则边
- **Best For**: Artisan、Fair Trade、Cultural Project
- **Do Not Use For**: SaaS、Corporate

## Steampunk

- **CSS Features**: 齿轮装饰 / 铜色 `#B87333` / 深棕 / 维多利亚装饰边
- **Best For**: Niche Gaming、Narrative Project、Themed Event
- **Do Not Use For**: 现代 SaaS、Mobile App
- **AI Prompt**: "steampunk aesthetic, Victorian brass gears, copper machinery, industrial ornaments"
