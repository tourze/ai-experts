# 多框架竞争分析

本 skill 提供跨框架交叉验证的竞争分析方法论。单一框架容易产生盲区，多个框架交叉印证能显著提高竞争判断的可靠性。

## 核心原则

- **不依赖单一框架**：任何单一框架都有盲点，至少用 3 个框架交叉验证。
- **框架之间必须验证同一结论**：不是每个框架各说各话，而是同一结论在多个框架中都能成立。
- **区分事实和判断**：框架输入是事实（市场份额、定价、功能列表），框架输出是判断。

## 可用框架矩阵

| 框架 | 适用阶段 | 核心问题 | 关注维度 |
|------|---------|---------|---------|
| Porter's Five Forces | 行业进入/退出 | 行业吸引力如何？ | 供应商/买方权力、替代威胁、竞争强度 |
| 3C Strategic Triangle | 战略定位 | 公司、客户、竞品的三角关系？ | 自身优势、客户需求、竞品差距 |
| Blue Ocean Strategy | 差异化 | 如何跳出红海？ | Eliminate-Reduce-Raise-Create |
| Strategy Clock | 定价策略 | 价格-价值定位？ | 低价/差异化/高价/混合 |
| BCG Matrix | 产品组合 | 资源如何分配？ | 市场增长率 × 相对份额 |
| SWOT | 全面扫描 | 优劣势与机会威胁？ | 内部 S/W + 外部 O/T |
| STP | 市场选择 | 卖给谁？ | 分群-目标-定位 |
| Competitive Intelligence | 情报收集 | 竞品在做什么？ | 功能/定价/渠道/人才动态 |
| Pricing Strategy | 定价验证 | 定价是否合理？ | 价值度量/竞品基准/弹性 |
| Crossing the Chasm | 扩张策略 | 如何跨越鸿沟？ | 创新者→早期采用者→早期多数 |
| SPACE Matrix | 战略态势 | 防守还是进攻？ | 财务/行业稳定性/竞争优势/环境 |
| Obviously Awesome | 定位方法 | 如何找到正确定位？ | 现有替代品分析 |

## 交叉验证流程

### Phase 1: 事实收集（不选框架）

先收集原始事实，再选框架分析。避免确认偏差。

- 竞品列表（直接/间接/潜在替代品）
- 各竞品的功能矩阵、定价、渠道、目标客户
- 市场规模和增长率数据来源
- 用户评价和 NPS 数据来源
- 竞品融资和团队信息来源

### Phase 2: 多框架并行分析

选择 3-5 个互补框架（不要选维度完全重叠的），每个框架独立得出结论：

```
事实 → [Porter's] → 结论 A
事实 → [3C]      → 结论 B
事实 → [SWOT]    → 结论 C
事实 → [Blue Ocean] → 结论 D
```

### Phase 3: 交叉印证

将各框架结论放在一起，寻找：

- **共识点**：多个框架得出相同结论（高置信度）
- **矛盾点**：不同框架得出矛盾结论（需要更深入调查）
- **盲区**：某框架发现但其他框架未覆盖的点（补充发现）

### Phase 4: 合成输出

只输出**经过交叉验证**的结论：

| 结论 | 验证框架 | 置信度 | 行动建议 |
|------|---------|--------|---------|
| ... | Porter's + 3C + SWOT | 高 | ... |
| ... | Blue Ocean only | 低 | 需要进一步验证 |

## 输出模板

```markdown
# 竞争分析：<scope>

## 分析范围
[分析对象 / 竞品列表 / 数据来源 / 时间范围]

## 框架选择与理由
| 框架 | 选择理由 |
|------|---------|

## 各框架结论
[每个框架一段，标注事实基础和判断]

## 交叉验证矩阵
| 结论 | Porter's | 3C | SWOT | Blue Ocean | 置信度 |
|------|----------|-----|------|-----------|--------|

## 高置信度发现
[多框架共识的结论]

## 矛盾与盲区
[需要进一步调查的点]

## 行动建议
[按置信度 × 影响力排序]

## 数据局限
[数据来源的覆盖范围和时效性]
```

## 红线

- **不用框架合理化已有结论**：如果结论先于分析得出，分析就是表演。
- **不堆框架**：3 个互补框架 > 10 个重叠框架。
- **不混淆框架输出和事实**：SWOT 中的 "Strength" 是判断不是事实，必须回溯到证据。
- **不做静态分析**：竞争态势是动态的，标注分析的时效性和需要重新评估的触发条件。
