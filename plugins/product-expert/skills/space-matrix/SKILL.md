# SPACE 矩阵

## 适用场景
- 战略方向选择：判断当前该进攻还是防守。
- 比 SWOT 更量化的战略态势评估。

## 核心约束
- 四维度：FS（财务优势）、CA（竞争优势）、ES（环境稳定性）、IS（产业优势）。
- **正负分方向是关键**：FS 和 IS 用正分（+1 最差到 +6 最好），CA 和 ES 用负分（-1 最好到 -6 最差）。搞反则战略判断南辕北辙。
- 向量计算：X = IS均值 + CA均值，Y = FS均值 + ES均值。箭头指向象限即为战略类型。
- 每个维度需 3-5 个指标，指标太少则样本不足、结论不稳。
- 四种结果：进取型（右上，积极扩张）、竞争型（右下，提升竞争力）、保守型（左上，聚焦核心）、防御型（左下，收缩剥离）。

## 不适用场景
- 缺乏财务数据：FS 和 CA 需要财务指标，数据不全时用定性的 [swot-analysis](../swot-analysis/SKILL.md)。
- 单产品/单市场：SPACE 假设有产业对比基准，极度垂直的小众市场缺乏可比数据。

## 代码模式

本 skill 是分析框架，不要求执行代码。输出时按“输入事实 → 框架拆解 → 关键判断 → 下一步动作”的结构组织，可用表格承载维度、证据、判断和建议。

## 检查清单
- [ ] 四个维度各有 3-5 个指标。
- [ ] 正负分方向没有搞反（FS/IS 正分，CA/ES 负分）。
- [ ] 向量计算正确，象限判断有依据。
- [ ] 战略建议与象限定位匹配。

## 反模式

### FAIL: 正负分搞反

```
CA（竞争优势）= +5（应该用负分）
→ 向量方向完全错误 → 战略判断南辕北辙
```

### PASS: 严格按规则评分

```
FS 均值 = +4.2, CA 均值 = -2.1, ES 均值 = -3.5, IS 均值 = +4.8
X = 4.8 + (-2.1) = 2.7（正，偏右）
Y = 4.2 + (-3.5) = 0.7（正，偏上）
→ 右上象限 → 进取型战略 → 积极扩张
```
