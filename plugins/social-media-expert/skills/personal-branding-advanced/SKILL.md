# 高级个人品牌建设

## 适用场景

- 账号已经有稳定输出，想从“会发内容”升级为“有清晰品牌识别”。
- 创始人、顾问、知识博主或垂类达人需要建立长期可复用的个人 IP。
- 用户想统一定位、视觉、表达方式和商业边界。
- 需要把内容定位反推到增长策略时，配合 [xiaohongshu-commercial-growth](../xiaohongshu-commercial-growth/SKILL.md)。
- 需要把品牌语气落到评论区和社群运营时，配合 [fan-operations](../fan-operations/SKILL.md)。

## 核心约束

- 品牌建设是“长期可重复识别”，不是一次性的头像、配色或口号。
- 先定义服务对象、核心观点和拒绝什么，再谈视觉风格。
- 不要同时经营多个相互冲突的人设；强品牌依赖长期一致性。
- 没有真实经历或可验证能力时，不要硬造“专家感”。
- 商业合作边界必须与品牌价值观一致，否则会迅速稀释信任。

## 代码模式

定位声明模板：

```text
我帮助 <目标人群>
通过 <独特方法>
解决 <核心问题>
获得 <结果>，
而我的差异化在于 <独特经历 / 判断标准 / 方法论>。
```

品牌资产清单模板：

```markdown
## 品牌锚点
- 一句话定位：
- 价值观：
- 擅长解决的问题：
- 明确不做什么：

## 识别要素
- 视觉主色：
- 标志性表达：
- 常用内容栏目：
- 高信任场景：
```

内容支柱设计模板：

```markdown
| 支柱 | 目标 | 常见形式 | 频次 |
| --- | --- | --- | --- |
| 专业判断 | 建立权威 | 观点型笔记/口播 | 每周 2 次 |
| 真实经历 | 建立信任 | 复盘/踩坑记录 | 每周 1 次 |
| 方法沉淀 | 建立复用 | 清单/模板/教程 | 每周 1 次 |
```

## 检查清单

- 已明确“为谁服务”和“拒绝什么合作”。
- 表达风格、封面风格和互动语气彼此一致。
- 品牌定位能反推出 3-5 个固定内容栏目。
- 品牌主张不会和当前变现路径冲突。
- 评论区、私域话术与 [fan-operations](../fan-operations/SKILL.md) 保持同一语气。
- 如果品牌目标是增长或变现，已同步给 [xiaohongshu-commercial-growth](../xiaohongshu-commercial-growth/SKILL.md)。

## 反模式

### FAIL: 视觉升级解决定位

```
“账号没什么人看 → 换头像 + 改配色 + 重做封面”
→ 视觉变了 / 定位仍模糊 / 流量仍 0
```

### PASS: 先定位再视觉

```
1. 服务谁：B 端创业者
2. 解决什么：从 0 到 1 找 PMF
3. 为何信我：3 次创业 / 2 次失败复盘
4. 拒绝什么：泛流量、付费课割韭菜
→ 定位清晰后再设计视觉
```

### FAIL: 摇摆人设

```
本周：”行业专家深度分析”
下周：”今日穿搭 + 美食探店”
→ 关注的人困惑 → 取消关注
```

### PASS: 三栏目稳定输出

```
| 栏目 | 频次 | 形式 |
| 专业判断 | 周 2 | 观点笔记 |
| 真实经历 | 周 1 | 复盘 |
| 方法沉淀 | 周 1 | 模板 |
→ 6 个月不变，建立可预期性
```
