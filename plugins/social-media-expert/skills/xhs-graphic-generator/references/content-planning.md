# 内容规划方法论

## 内容分析流程

### 1. 识别内容类型

| 类型 | 特征 | 示例 |
|------|------|------|
| 技巧分享 | 3-5个要点，实操性强 | 拍照技巧、化妆步骤 |
| 知识科普 | 概念解释+应用场景 | 护肤成分、理财入门 |
| 经验总结 | 时间线或心路历程 | 入坑分享、踩坑避雷 |
| 产品测评 | 对比维度+结论 | 好物推荐、横评 |

### 2. 确定内容密度

**轻量级**（5-7张，60-100字/张）
- 单一主题，要点清晰
- 适合：技巧、穿搭、日常分享

**中等级**（7-12张，100-180字/张）
- 多维度展开，有深度
- 适合：攻略、科普、测评

**重量级**（10-18张，150-250字/张）
- 系统性内容，信息密集
- 适合：教程、深度分析、长文改写

### 3. 规划图片序列

**封面图（1张）**
- 吸睛标题 + 视觉冲击
- 点明主题，制造好奇

**内容图（N张）**
按逻辑组织：
- 痛点共鸣 → 解决方案
- 概念引入 → 详细拆解
- 时间顺序 → 步骤流程
- 对比展示 → 结论总结

**收尾图（1张）**
- 金句总结
- 互动引导（评论、收藏、关注）

## 图片内容分配原则

### 每张图的信息量

- **标题**：5-15字，一眼抓住重点
- **正文**：2-4个要点，每点1-2行
- **装饰**：不超过3种元素

### 视觉节奏

```
封面：视觉冲击 ████████████ 100%
内容1：高信息 ████████░░ 80%
内容2：中信息 ██████░░░░ 60%
内容3：高信息 ████████░░ 80%
收尾：情感共鸣 ██████████ 90%
```

## 文案改写技巧

### 从长文到卡片

1. **提取核心观点**：每张图只承载1个核心信息
2. **口语化改写**：书面语 → 对话感
3. **增加情绪词**：「超绝」「救命」「真的会谢」
4. **善用符号**：✨ 💡 ❶❷❸ → ✓✗

### 标题公式

- 数字+痛点：「6个习惯让你早起不痛苦」
- 对比反差：「从废片到神图只需6秒」
- 悬念疑问：「为什么你的照片总是不好看？」
- 情绪共鸣：「终于有人把XX讲清楚了！」
