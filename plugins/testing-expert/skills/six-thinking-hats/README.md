# six-thinking-hats

六顶思考帽测试法的配套说明文档。

## 包含内容

- [SKILL.md](./SKILL.md)：给模型的执行说明
- [solo-session-template.md](./resources/templates/solo-session-template.md)：个人分析模板
- [team-session-template.md](./resources/templates/team-session-template.md)：团队会议模板
- [api-testing-example.md](./resources/examples/api-testing-example.md)：API 测试策略示例

## 使用建议

- 个人快速分析时优先使用 `solo-session-template.md`
- 团队对齐时优先使用 `team-session-template.md`
- 先读 `SKILL.md` 再选模板，避免混帽
