# Contract Matrix

这份矩阵用于在起草录用通知、劳动合同和补充协议时，快速核对每类模板的必填字段与本地化提醒。

## Offer Letter

| 字段 | 必填 | 说明 |
|------|------|------|
| 候选人姓名 | 是 | 与身份证明/护照保持一致 |
| 职位名称 | 是 | 与内部编制名称、职级口径一致 |
| 汇报线 | 是 | 写到岗位而不是只写人名 |
| 用工类型 | 是 | employee / contractor / intern |
| 生效条件 | 是 | 背调、身份校验、签署附属文件 |
| 接受期限 | 是 | 避免长期悬而未决 |

## Employment Agreement

| 模块 | 必填 | 本地化提示 |
|------|------|-----------|
| 工资与支付周期 | 是 | 币种、税前/税后、结算周期 |
| 试用期 | 视法域 | 一些法域限制试用期时长 |
| 工时与加班 | 是 | exempt / non-exempt 要分开写 |
| 保密与知识产权 | 是 | 既有成果保留条款要单列 |
| 终止与通知期 | 是 | 按法域写，不要跨法域复制 |
| 争议解决 | 视法域 | 仲裁地、法院、适用法要一致 |

## Amendment / Addendum

| 场景 | 额外字段 |
|------|----------|
| 远程办公附录 | 工作地点、设备、报销、安全要求 |
| 调岗调薪 | 生效日期、岗位职责、薪酬调整依据 |
| 保密补充协议 | 信息范围、返还义务、例外情形 |
