# 四文件模板

`persistent-planning` 建议使用四个文件持久化任务状态。

## context.md

```markdown
# Context
- Task Statement:
- Desired Outcome:
- Intent Hypothesis:
- Unknowns:
- Decision Boundaries:
- Touchpoints:
```

## task_plan.md

```markdown
# Task Plan
| Phase | Status | Deliverable |
| --- | --- | --- |
| Discovery | pending | ... |
| Planning | pending | ... |
| Implementation | pending | ... |
| Verification | pending | ... |
| Delivery | pending | ... |
```

## findings.md

```markdown
# Findings
- Source:
- Key Evidence:
- Trust Level:
- Notes:
```

## progress.md

```markdown
# Progress Log
- 2026-01-01 10:00: init context
- 2026-01-01 10:30: phase discovery complete
```
