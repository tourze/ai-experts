# Windows UI Automation 威胁模型

## 核心资产

- 凭据与令牌：密码框、验证码、会话令牌、API Key
- 敏感界面：安全对话框、管理工具、企业内部业务系统
- 系统完整性：高权限窗口、系统快捷键、自动化脚本本身

## 典型威胁

- 跨完整性级别自动化高权限窗口，造成提权链路。
- 读取敏感控件值、截图或剪贴板内容，造成信息泄露。
- 通过热键或消息注入绕过人工确认。
- 没有速率限制或停止条件，造成脚本失控和界面拒绝服务。
