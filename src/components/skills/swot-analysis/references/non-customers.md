# 非顾客三层分析

## 适用场景

- 需要从现有客户之外寻找需求空间。
- 蓝海战略中需要解释新价值曲线服务谁。
- 现有市场增长见顶，需要理解拒绝购买的人。

## 三层非顾客

| 层级 | 描述 | 研究问题 |
| --- | --- | --- |
| 第一层 | 勉强使用现有方案，随时可能离开 | 他们忍受了什么？ |
| 第二层 | 明确拒绝现有方案 | 为什么拒绝？成本、复杂度、信任还是渠道？ |
| 第三层 | 从未被行业触达 | 他们如何替代解决？有没有完全不同场景？ |

## 访谈方向

- 你为什么没有购买或继续使用现有方案？
- 你现在如何解决这个问题？
- 你最不能接受的成本或复杂度是什么？
- 如果只保留一个能力，什么会让你愿意尝试？

## 输出要求

- 非顾客不要泛称“所有未购买者”，必须分层。
- 每层至少写出阻力、替代方案和可能的新价值。
- 结论要能回到 ERRC 或策略画布。
