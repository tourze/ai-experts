## 反模式

### FAIL: ICP = 所有人

```
ICP："任何想提升效率的中小企业"
→ 销售名单 = 所有公司
→ 转化率 1%，CAC 高得离谱
```

### PASS: 窄而深

```
ICP（核心）：
- 公司：中文区电商，年 GMV 500w-5000w，员工 20-100
- 买方：运营总监（负责数据看板）
- 触发：刚完成融资 / 新增渠道
- 排除：传统零售 / 海外品牌 / 员工 < 20
→ 销售名单从 10 万缩到 800
→ 转化率 8%，CAC 降低 60%
```

### FAIL: 只人口统计

```
ICP："25-40 岁，月收入 1w+，一线城市"
→ 不知道为什么会买
→ 营销投放精准但转化低
```

### PASS: 加触发 + 痛点

```
人口（基线）：25-40 岁，月入 1w+
触发事件：
- 刚换工作（3-6 个月内）
- 刚结婚 / 刚有娃
- 刚搬家
痛点：搬家后通勤时长 +40 分钟
→ 投放定向："换工作 + 通勤 > 1h"
```
