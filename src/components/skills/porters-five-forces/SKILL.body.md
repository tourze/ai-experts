## 反模式

### FAIL: 只看现有竞品

```
"竞争激烈" → 列 5 家竞品名字
→ 漏看：
- AI 工具能否替代整个类别（替代品）
- 大客户能否自建（买方议价）
- 关键供应商断供（供应商议价）
- 新进入者门槛是否在降低
```

### PASS: 五力都过

```
| 力 | 强弱 | 驱动 | 5 年趋势 |
| 现有竞争 | 高 | 同质化 | 持平 |
| 替代品 | 极高 | LLM 出现 | 上升快 |
| 买方议价 | 中 | 客户集中度提高 | 上升 |
| 供应商议价 | 高 | 上游云成本 | 持平 |
| 新进入者 | 中 | 技术门槛降低 | 上升 |
→ 真正的威胁来自替代品而非现有竞品
```

### FAIL: 行业 vs 执行混淆

```
"我们利润低 → 五力分析说行业差"
→ 实际：行业平均毛利 70%，自家是 35% → 是执行问题
→ 用五力解释执行问题 → 错误归因
```

### PASS: 区分

```
先看：行业平均利润率 vs 自家利润率
- 自家 ≥ 平均：你在跑赢，五力相对友好
- 自家 < 平均：执行问题，先修内部
- 平均 < 10%：行业结构问题，五力分析有意义
```
