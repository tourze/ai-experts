## 适用场景
- 新行业进入评估、市场格局判断、战略机会复盘。
- 与内部能力/执行风险结合分析时，可配合 [swot-analysis](../swot-analysis/SKILL.md) 或 [competitive-teardown](../competitive-intelligence/SKILL.md)。
- 补充分析框架：[references/3c-strategic-triangle.md](references/3c-strategic-triangle.md) — 3C 战略三角（客户-公司-竞争对手）；[references/strategy-clock.md](references/strategy-clock.md) — 战略钟（定价-价值定位）。

## 核心约束
- 五力分析关注行业结构，不是单个竞争对手花活。
- 每一力都要说明强弱原因、时间尺度和对盈利能力的影响。
- 不要把“市场大”误当作“行业结构有利”。

## 代码模式
```markdown
| 力量 | 当前强弱 | 驱动因素 | 对利润的影响 |
| --- | --- | --- | --- |
```

## 检查清单
- [ ] 五种力量都已覆盖，没有只讲竞争对手。
- [ ] 结论解释了行业利润空间与进入门槛。
- [ ] 已指出哪些力量可通过战略动作改善。
- [ ] 与目标公司的资源与定位判断能对上。

## 反模式

### FAIL: 只看现有竞品

```
"竞争激烈" → 列 5 家竞品名字
→ 漏看：
- AI 工具能否替代整个类别（替代品）
- 大客户能否自建（买方议价）
- 关键供应商断供（供应商议价）
- 新进入者门槛是否在降低
```

### PASS: 五力都过

```
| 力 | 强弱 | 驱动 | 5 年趋势 |
| 现有竞争 | 高 | 同质化 | 持平 |
| 替代品 | 极高 | LLM 出现 | 上升快 |
| 买方议价 | 中 | 客户集中度提高 | 上升 |
| 供应商议价 | 高 | 上游云成本 | 持平 |
| 新进入者 | 中 | 技术门槛降低 | 上升 |
→ 真正的威胁来自替代品而非现有竞品
```

### FAIL: 行业 vs 执行混淆

```
"我们利润低 → 五力分析说行业差"
→ 实际：行业平均毛利 70%，自家是 35% → 是执行问题
→ 用五力解释执行问题 → 错误归因
```

### PASS: 区分

```
先看：行业平均利润率 vs 自家利润率
- 自家 ≥ 平均：你在跑赢，五力相对友好
- 自家 < 平均：执行问题，先修内部
- 平均 < 10%：行业结构问题，五力分析有意义
```
