# Campaign Metrics Benchmarks

## Core Formulas

| Metric | Formula | Check |
| --- | --- | --- |
| ROI | `(revenue - spend) / spend` | Revenue and spend use the same currency and time window. |
| ROAS | `revenue / spend` | Revenue is attributed with the stated model. |
| CPA | `spend / acquisitions` | Acquisition event is deduplicated. |
| CPL | `spend / leads` | Lead quality is not assumed from volume alone. |
| CAC | `sales + marketing cost / new customers` | Includes the full cost base, not only media spend. |
| CVR | `conversions / sessions or clicks` | Denominator matches the channel reporting source. |

## Benchmark Discipline

- Prefer the account's own trailing baseline over generic industry ranges.
- Split benchmarks by channel, geo, device, audience, and funnel stage before judging performance.
- A metric is only actionable when paired with sample size and confidence.
- Use external benchmarks only as a sanity check, not as the target itself.

## Red Flags

- ROAS improves while new customers fall.
- CPA improves after a tracking change with no corresponding funnel change.
- Paid platform conversions exceed site-side conversions without a stated view-through window.
- One channel looks efficient only under one attribution model.
