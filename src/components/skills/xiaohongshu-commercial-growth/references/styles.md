# 小红书图文风格方案

## 适用场景

- 需要为封面、知识卡、清单卡选择视觉风格。
- 需要统一系列内容的颜色、字体、布局和品牌识别。
- 需要把商业目标转成更适合小红书的视觉表达。

## 常见风格

| 风格 | 适合内容 | 特征 |
| --- | --- | --- |
| 清爽知识卡 | 教程、清单、方法论 | 白底、细线、少色、编号清晰 |
| 温柔生活感 | 美妆、穿搭、家居、母婴 | 暖色、纸感、轻阴影、柔和图片 |
| 高对比封面 | 商业、职场、效率 | 大标题、强色块、少元素 |
| 手账拼贴 | 旅行、生活方式、个人 IP | 贴纸、手写感、照片叠放 |
| 专业报告感 | B2B、知识付费 | 图表、网格、冷静配色 |

## 选择规则

- 商业转化强：标题更直接，对比更强，CTA 明确。
- 人设经营：保持固定色系、固定标题位置和固定语气。
- 知识内容：优先可读性，不要牺牲文字清晰度。
- 系列化：封面结构固定，只换主题色和核心图。

## 输出要求

- 给出风格名、色板、字体方向、布局规则和禁用元素。
- 说明为什么该风格匹配目标人群和商业路径。
