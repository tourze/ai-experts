## 反模式

### FAIL: 先投放后补承接

```
“先用 1w 元投流测试 → 再看效果”
→ 流量进来 → 主页空 / 客服没接 / 私域没准备
→ 投放预算白烧
```

### PASS: 先承接再开闸

```
1. 主页置顶 + 个签清晰
2. 内容池 ≥ 10 篇高质量
3. 私域承接流程跑通
4. 自然流转化率 > 1%
5. 才开始付费投放
```

### FAIL: 通用方案套所有

```
品牌 / 电商 / 知识 IP / 本地店 → 同一套”小红书爆款方法论”
→ 哪类都跑不通
```

### PASS: 按业务定制

```
品牌：种草 + 心智沉淀，不追单条爆
电商：搜索流量 + 店铺承接 + 复购
知识 IP：深度内容 + 私域转化 + 高客单
本地店：地理位置 + 到店核销
→ 各自的指标体系完全不同
```
