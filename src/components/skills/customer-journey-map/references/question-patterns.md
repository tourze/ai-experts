# Mom Test 问题模式

## 适用场景

- 需要把“你会不会用”改成过去式事实问题。
- 需要挖掘当前替代方案、真实频率、成本和决策链。
- 需要准备创业想法验证、流失访谈或 B2B 发现访谈提纲。

## 好问题模式

### 过去事实

```text
你上次遇到这个问题是什么时候？
当时你具体怎么处理？
花了多长时间或多少钱？
```

### 当前替代方案

```text
你现在用什么办法解决？
这个办法最麻烦的部分是什么？
你为它付过钱、投入过人力或搭过流程吗？
```

### 追问具体例子

```text
能打开上一次的记录给我看吗？
上次是谁参与决策？
最后为什么没有继续推进？
```

### 购买与推进

```text
如果要解决这个问题，预算来自哪里？
谁需要批准？
下一步你愿意引入谁一起聊？
```

## 禁问清单

- “如果我们做 X，你会用吗？”
- “这个想法好不好？”
- “你愿意付多少钱？”但没有真实购买上下文。
- “你觉得别人会不会需要？”

## 输出要求

- 每个问题都能产生事实、行为、成本或下一步承诺。
- 每个假设至少配 2-3 个可追问问题。
- 访谈后标注证据强度，而不是只摘正面语句。
