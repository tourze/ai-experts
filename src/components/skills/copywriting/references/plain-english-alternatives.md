# Plain English Alternatives

## Replacement Rules

| Weak Phrase | Replace With |
| --- | --- |
| 提供卓越体验 | Say what improves, for whom, and by how much. |
| 助力企业增长 | Name the workflow, metric, or business result. |
| 一站式解决方案 | List the concrete jobs handled in one place. |
| 深度赋能 | Use a verb: automate, reduce, detect, approve, reconcile. |
| 革命性 / 颠覆性 | Use proof or remove it. |

## Editing Pattern

```text
Abstract claim -> Concrete action -> Buyer-visible result -> Proof
```

## Checklist

- Replace vague nouns with verbs.
- Replace adjectives with numbers, examples, or evidence.
- Use customer language from calls, reviews, support tickets, or search terms.
- Keep necessary qualifiers; do not simplify away truth.
