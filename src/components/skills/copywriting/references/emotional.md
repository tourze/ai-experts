# Emotional

## Emotional Drivers

| Driver | Copy Direction |
| --- | --- |
| Relief | Remove recurring friction or anxiety. |
| Control | Show visibility, predictability, and fewer surprises. |
| Status | Help the buyer look competent to peers or leadership. |
| Belonging | Show similar teams making the same move. |
| Progress | Make the next step feel achievable. |

## Rules

- Emotion should come from the user's situation, not exaggerated adjectives.
- Pair emotional language with concrete proof.
- Avoid fear-based pressure in sensitive contexts.
- Name the cost of inaction only when it is real.
