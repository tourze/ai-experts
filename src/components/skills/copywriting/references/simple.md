# Simple

## Goal

Make the reader remember one useful idea.

## Process

1. List all claims.
2. Choose the one claim the page must make.
3. Remove secondary claims from the headline.
4. Move proof and nuance into supporting sections.
5. Test whether a reader can repeat the idea after 5 seconds.

## Pattern

```text
For {{specific audience}}, {{product}} helps {{job}} so {{outcome}}.
```

## Check

- One audience.
- One job.
- One outcome.
- No category buzzwords unless they help the buyer compare.
