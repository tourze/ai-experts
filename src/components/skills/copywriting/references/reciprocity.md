# Reciprocity

## Use Cases

- Free checklist, audit, calculator, teardown, trial, or diagnostic.
- Educational email before a sales ask.
- Landing page that gives a useful sample before asking for a meeting.

## Rules

- Give something genuinely useful, not a disguised brochure.
- Make the next step proportional to the value already given.
- State the boundary: what is free, what requires signup, what happens next.
- Do not imply an obligation the user did not accept.

## Copy Pattern

```text
Get {{useful asset}}.
Use it to {{specific job}}.
If it shows {{problem}}, book {{low-friction next step}}.
```
