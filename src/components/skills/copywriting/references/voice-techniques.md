# Voice Techniques

## Voice Dimensions

| Dimension | Choices |
| --- | --- |
| Directness | blunt, concise, explanatory, consultative |
| Energy | calm, sharp, warm, urgent |
| Proof style | numbers, examples, demos, customer quotes |
| Risk posture | conservative, challenger, expert, practical |

## Technique

- Give the brand a point of view, not just a tone adjective.
- Use sentence length to control pace.
- Put concrete nouns near the start of sentences.
- Replace "we help you" with the user action or outcome.
- Keep one deliberate rough edge when everything sounds too polished.
