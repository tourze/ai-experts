## 使用场景

- 新创业想法的首次系统化评估
- 季度复盘检查假设漂移
- 投资人沟通前的自检
- Pivot 决策前的全盘重估

## 12 维度评估

### 1. 想法验证

- [ ] 问题是否被真实用户口头确认（不是"我觉得"）？
- [ ] 用户当前如何解决这个问题？（替代方案）
- [ ] 不解决的后果是什么？（维生素 vs 止痛药）
- [ ] 是否做过 Mom Test 访谈？结论是什么？

### 2. 理想客户画像（ICP）

- [ ] 是否能明确描述"最值得卖"的客户画像？
- [ ] ICP 是否有共同的行为特征（不只是人口统计学）？
- [ ] ICP 的获取渠道是否可触达？
- [ ] 购买决策者与使用者是否一致？

### 3. 市场规模

- [ ] TAM/SAM/SOM 是否用至少两种方法交叉验证？
- [ ] 市场是否在增长？增长率是多少？
- [ ] 市场中是否有足够的头部客户支撑收入目标？
- [ ] 地域扩张路径是否清晰？

### 4. 商业模式

- [ ] 收入模型是什么？（订阅/交易/广告/许可/服务）
- [ ] 单位经济（unit economics）是否正向？
- [ ] 客户生命周期价值（CLV）是否 ≥ 3× CAC？
- [ ] 毛利率是否足以支撑增长投入？

### 5. 业务健康度

- [ ] 关键指标是什么？（MRR/DAU/留存率/NPS）
- [ ] 月度增长率是否达到"好"的基准？
- [ ] 流失率是否在可接受范围内？
- [ ] 净收入留存率（NRR）是否 > 100%？

### 6. 定价

- [ ] 定价基于价值还是成本？
- [ ] 是否做过价格敏感度测试？
- [ ] 定价与客户支付意愿是否匹配？
- [ ] 免费层/试用期的转化率是多少？

### 7. 渠道

- [ ] 主要获客渠道是什么？是否验证过？
- [ ] 渠道单位经济（CAC）是否可接受？
- [ ] 是否有 2+ 个获客渠道，避免单点依赖？
- [ ] 渠道与 ICP 是否匹配？

### 8. 竞争

- [ ] 前 3 名竞品是谁？各自优势？
- [ ] 差异化是基于功能、体验、价格还是网络效应？
- [ ] 竞品是否有不可逾越的壁垒（数据/牌照/专利）？
- [ ] 用户为什么选你不选竞品？（一句话回答）

### 9. 融资

- [ ] 当前阶段需要多少资金？用途拆分？
- [ ] 融资轮次与里程碑是否对齐？
- [ ] 是否有 3+ 个潜在投资人线索？
- [ ] 是否有清晰的退出路径或盈利路径？

### 10. 团队

- [ ] 核心团队是否覆盖关键技术/市场/运营角色？
- [ ] 团队是否有相关行业经验？
- [ ] 关键岗位是否有备选方案（bus factor）？
- [ ] 团队是否有共同愿景和冲突解决机制？

### 11. 用户旅程

- [ ] 从认知到付费的完整路径是否清晰？
- [ ] 每个环节的转化率是否已知？
- [ ] 最大流失点在哪里？原因是什么？
- [ ] 是否有 Aha Moment 设计？

### 12. 不确定性

- [ ] 最大的 3 个不确定性是什么？
- [ ] 每个不确定性是否有验证计划和时间节点？
- [ ] 如果最关键假设被证伪，pivot 方向是什么？
- [ ] 现金流跑道还有多少个月？

## 输出模板

```markdown
# 创业项目评估：<项目名>

## 总评
[可行性评级：Strong / Moderate / Weak]
[致命盲区：...]
[核心优势：...]

## 维度评分
| 维度 | 状态 | 置信度 | 关键发现 |
|------|------|--------|---------|
| 想法验证 | ✅/⚠️/❌ | 高/中/低 | ... |

## 致命假设（必须验证）
1. ...
2. ...

## 优先行动（Top 3）
1. ...
2. ...
3. ...

## 不做的事
[当前阶段不需要投入的维度]
```

## 红线

- **不跳过用户验证**：没有真实用户反馈的想法评估 = 自我欺骗。
- **不做线性外推**：市场规模不用"假设 1% 市占率"这种无意义估算。
- **不回避竞争**："我们没有竞争对手"要么是没做调研，要么是没市场。
- **不混淆假设和事实**：清单中每项标注"已验证"还是"假设"。
