# 生成式工具创意工作流

## 适用场景

- 需要批量生成广告角度、文案变体、图片 prompt 或视频脚本。
- 已有高低表现素材，需要系统化迭代而不是随机改词。
- 需要把创意 brief 转成可测试变量矩阵。

## 工作流

1. 输入平台、版位、受众、转化目标和禁用词。
2. 拆出 3-5 个创意角度：痛点、收益、对比、风险、社会证明。
3. 每个角度生成标题、主文案、CTA 和素材 prompt。
4. 按平台规格裁剪长度和画面比例。
5. 保留变量名，便于投放复盘。

## Prompt 模板

```text
平台：[Meta/LinkedIn/TikTok]
受众：[角色 + 痛点]
目标：[转化动作]
角度：[ROI/效率/风险/社会证明]
限制：[字符、禁用词、语气]
输出：[标题 10 条 + 主文案 5 条 + CTA 3 条 + 图片 prompt 3 条]
```

## 质量检查

- 每组变体只改一个主要变量。
- 不生成夸大承诺、违规对比或无法证明的数据。
- 图片 prompt 写清主体、场景、文案、构图和负面约束。
