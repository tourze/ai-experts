- [ ] 域名已在商务管理平台验证
- [ ] 已配置汇总事件测量

- [ ] Web 事件已为 iOS 14+ 跟踪设置优先级

### 受众设置

- [ ] 已创建自定义受众：
  - 网站访客（全部，30/60/90/180 天）
  - 关键页面访客
  - 视频观众（25%、50%、75%、95%）
  - 主页/Instagram 互动者
  - 已上传客户列表
- [ ] 已创建类似受众（1%、1-3%）
- [ ] 为常见的定向条件保存了受众

### 目录（电商）

### 创意素材

- [ ] 图片尺寸正确：

- [ ] 视频格式正确
- [ ] 广告文案变体已准备就绪
- [ ] 所有目标网址中包含 UTM 参数

### 合规

- [ ] 已声明特殊广告类别（如涉及住房、信贷、就业、政治）

- [ ] 预算设置正确（每日 vs. 总预算）

- [ ] 定向条件匹配目标受众
- [ ] 广告创意已审批
- [ ] 团队已收到上线通知
- [ ] 报表看板已准备就绪
