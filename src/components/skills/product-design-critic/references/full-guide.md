# 想法验证完整指南

## 目标

在投入大量开发和推广前，用最小成本验证“问题是否真实、用户是否愿意付费、方案是否可行”。

## 四步法

1. 问题验证：是否高频、刚需、可量化。
2. 用户验证：目标人群是否准确、是否愿意切换。
3. 方案验证：是否显著优于替代方案。
4. 商业验证：是否有可持续获客和变现路径。

## 证据分层

- A 级：真实行为数据（付费、留存、复购）
- B 级：强意向信号（预约、试用、押金）
- C 级：口头反馈（访谈、问卷）

## Kill Criteria 示例

- 60 天内付费用户 < 10：重新定位
- 90 天内月收入 < $1k：停止投入
- D30 留存 < 20%：不继续放大投放
