# 客户与产品生命周期策略矩阵

## 价值分层（顾客金字塔）

| 层 | 信号 | 服务策略 |
|----|------|----------|
| 铂金（≈10%）| 贡献 50–60% 利润 | 专属 CSM、季度业务评审、定制方案 |
| 金（≈20%） | 贡献 20–30% 利润 | 共享 CSM 团队、月度 check-in |
| 铁（≈40%） | 贡献 10–20% 利润 | 自助平台 + 知识库 + AI 客服 |
| 铅（≈30%） | 利润为负 | 提价 / 限制服务 / 不续约 |

## 阶段策略

| 轴 | 阶段 | 营销重点 |
|----|------|----------|
| 产品 PLC | 导入 | 核心版本 + 教育市场 + 选择性分销，找可复制的增长模式 |
| 产品 PLC | 成长 | 扩功能 + 品牌 + 扩覆盖，抢市场份额 |
| 产品 PLC | 成熟 | 差异化改良 + 促销/忠诚 + 全渠道，布局下一代 |
| 产品 PLC | 衰退 | 精简 SKU + 降价 + 收缩渠道 + 最小投入，引导迁移 |
| 客户 LC | 观望 | 教育 + 信任建设 |
| 客户 LC | 激活 | 引导上手 + aha moment |
| 客户 LC | 扩展 | 升级 / 交叉销售 |
| 客户 LC | 续约 | 价值回顾 + 续约推荐 |
| 客户 LC | 赢回 | 流失原因诊断 + 针对性挽回 |

## 反模式示例

### FAIL: 所有客户一视同仁

```
所有客户享受同样的服务标准
-> 铂层觉得"花这么多钱没特殊待遇"
-> 铅层消耗了不成比例的服务资源
```

### PASS: 差异化服务

```
铂层（50 家，贡献 60% 利润）：每家配专属 CSM，季度业务评审
金层（200 家，贡献 25% 利润）：共享 CSM 团队，月度 check-in
铁层（1000 家，贡献 15% 利润）：自助平台 + 知识库 + AI 客服
铅层（300 家，-5% 利润）：下季度提价 30%，不接受则不续约
```

### FAIL: 衰退期还在大量投入

```
产品月活连续 6 个月下降 -> 继续投入大量广告预算
-> 应该：减少投入，收割现金，引导客户迁移到新产品
```

### PASS: 阶段匹配的产品组合

```
产品 A（成熟期）：停止新功能投入，优化利润率，现金流输血新产品
产品 B（成长期）：追加 50% 预算，抢市场份额
产品 C（导入期）：小规模测试，找到 PMF 再加注
```
