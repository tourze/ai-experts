## 不适用场景
- 客户数 < 100：样本不足以分层，直接逐个客户管理。
- 纯 PLG / 自助产品（无客户经理）：用产品内分群替代人工差异化服务。
- 多产品组合资源分配：用 `bcg-matrix`。
- 增长拐点判断：用 `s-curve-growth`。

## 检查清单
- [ ] 分层依据是利润贡献（CLV），不是收入。
- [ ] 铅层有明确处理方案（不是忽略）。
- [ ] 做了金字塔健康度评估（头部依赖、铅层占比、流动性）。
- [ ] 准确判断了产品所处 PLC 阶段，策略与阶段匹配。
- [ ] 考虑了下一阶段过渡准备（成熟期前布局下一代）。
- [ ] 产品 PLC 与客户 LC 分别分析，没有混淆。
