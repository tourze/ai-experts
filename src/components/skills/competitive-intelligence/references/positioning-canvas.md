# Positioning Canvas

## Flow

```text
Competitive alternatives -> Unique attributes -> Customer value -> Best-fit customers -> Market category
```

## Canvas

| Field | Question |
| --- | --- |
| Competitive alternatives | What would customers use if we did not exist? |
| Unique attributes | What capabilities do we have that alternatives do not? |
| Value | Why do those attributes matter in business terms? |
| Best-fit customers | Who cares most and can buy now? |
| Market category | What frame makes the value obvious? |
| Proof | What evidence makes the claim believable? |

## Rules

- Start from the customer's current workaround, not your feature list.
- Keep the first target segment narrow.
- Choose a category that helps buyers compare correctly.
- Update messaging, sales qualification, and roadmap tradeoffs together.
