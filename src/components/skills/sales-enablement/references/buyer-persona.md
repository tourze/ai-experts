# 买家画像卡

每个角色一张卡：

| 字段 | 说明 |
|------|------|
| 角色 | 职位 + 汇报线 |
| 核心目标 | 今年最重要的 2-3 个 KPI |
| 主要痛点 | 阻碍目标达成的障碍 |
| 常见异议 | 最可能提出的反对意见 |
| 评估标准 | 选型时看什么 |
| 购买流程 | 在决策链中的角色和影响力 |
| 沟通角度 | 跟这个人讲什么最有效 |

## 按买家类型定制
- **技术买家**：架构适配、安全合规、集成成本。
- **经济买家**：ROI、总体拥有成本、回收周期。
- **内部倡导者**：对比表、一句话价值、审批模板。
- **终端用户**：易用性、学习曲线、日常体验。
