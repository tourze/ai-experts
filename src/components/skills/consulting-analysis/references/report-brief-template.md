# 咨询分析 brief 与报告骨架

## 输入 brief

```json
{
  "研究主题": "中国企业知识管理软件市场",
  "分析目标": ["市场规模", "竞争格局", "客户需求", "进入建议"],
  "交付物": "咨询级 Markdown 报告",
  "数据要求": ["官方统计", "公司财报", "产品定价页", "用户访谈摘录"]
}
```

## 报告骨架

```markdown
# 执行摘要
## 1. 问题定义
## 2. 研究方法与数据源
## 3. 核心发现
## 4. 战略含义
## 5. 建议与下一步
```

## 使用要求

- 先补齐研究对象、目标读者、时间范围和地区范围。
- 每个章节要绑定分析模型和数据来源。
- 核心发现必须能回溯到证据、方法和推理链路。
