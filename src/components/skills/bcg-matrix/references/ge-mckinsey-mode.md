# GE-McKinsey 九宫格模式

当 BCG 2x2 太粗、需要量化打分时切到此模式。

## 轴定义

- X 轴竞争实力（内部）、Y 轴市场吸引力（外部），都是多指标加权得分。
- 内外部维度至少各用 5 个指标。
- 打分差异 > 1 分才有区分度。

## 权重原则

权重不能平均分配，必须反映行业关键成功因素。

- SaaS：市场增长率 30%、NRR 天花板 25%，因为增长是 SaaS 估值核心。
- 制造业：盈利性 30%、进入壁垒 25%，因为制造业资本密集。
- 快消品：市场规模 30%、竞争强度 25%，因为渠道是关键约束。

## 使用规则

- 九宫格结论是战略方向，不是精确数字；争论 3.2 还是 3.3 没有意义。
- 与 BCG 2x2 结果交叉验证：分类一致提升信心，分歧暴露被忽略的指标。
- 输出必须转成投资、选择性投入、维持、收缩或退出动作。
