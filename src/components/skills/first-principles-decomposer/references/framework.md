# 第一性原理框架 — 详细方法论

## 伊隆·马斯克方法（物理学思维）

"把事物归结为最基本的真理，然后从那里向上推理。"

### 电池示例
- **传统思维**："电池很贵（600 美元/kWh），一直如此"
- **第一性原理**：电池由什么组成？碳、镍、铝、聚合物
- **原子级问题**：这些材料在伦敦金属交易所的价格是多少？
- **发现**：原材料成本 = 80 美元/kWh
- **重构**：设计更好的制造工艺 → 特斯拉的成本突破

## 分解阶梯

### 第 1 层：表面问题
"我们需要更好的家长互动"

### 第 2 层：功能需求
"家长需要感觉与孩子的日常有联系"

### 第 3 层：情感核心
"家长对无法陪伴感到内疚"

### 第 4 层：基本事实
"人类需要他们所爱之人安好的证据"

### 第 5 层：原子级解决方案
照片 + 时间戳 + 孩子的脸 = 安好证明
其他都是装饰。

## 常见假设陷阱

### "行业标准"陷阱
"大家都这么做" → 为什么大家都这么做？
通常是：惯性，而非逻辑

### "客户说"陷阱
"客户要求 X" → 他们试图解决什么_问题_？
通常是：他们在描述方案，而非需求

### "技术存在"陷阱
"我们应该用 [技术] 因为它存在" → 这满足了什么基本需求？
通常是：方案在寻找问题

### "竞争对手这么做"陷阱
"对手有这个功能" → 它真的解决实际问题吗？
通常是：功能膨胀，而非价值

## 能切中要害的问题

1. "如果今天从零开始，我们会这样构建吗？"
2. "一个完全不了解情况的人会觉得这里面有什么奇怪的地方？"
3. "如果这个成本贵 10 倍，我们还会做吗？"
4. "如果这是非法的，我们会怎么做？"
5. "这个的版本能让一个孩子理解吗？"

## 与其他技能的结合

- 使用**逆向思维**来验证："这个基础方案会如何失败？"
- 使用**二阶后果**来推演："我们实施之后会发生什么？"
- 使用**事前验尸**在构建前："想象这个已经失败了。为什么？"
