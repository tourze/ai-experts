## 代码模式

### PESTEL 分析模板

```markdown
| 维度 | 关键因素 | 影响方向 | 时间窗 | 量级 | 行动含义 |
|---|---|---|---|---|---|
| Political（政治） | | ↑利好/↓利空/↔不确定 | 短/中/长 | 高/中/低 | |
| Economic（经济） | | | | | |
| Social（社会） | | | | | |
| Technological（技术） | | | | | |
| Environmental（环境） | | | | | |
| Legal（法律） | | | | | |
```

### 各维度常见扫描点

```markdown
Political（政治）
- 政府产业政策、补贴方向
- 国际贸易关系、制裁和关税
- 政治稳定性、政权更迭风险
- 税收政策变化

Economic（经济）
- GDP 增速、通胀率、利率
- 汇率波动（跨境业务）
- 消费者信心指数、购买力
- 融资环境（VC 活跃度、估值倍数）

Social（社会）
- 人口结构变化（老龄化、Z 世代）
- 消费习惯和价值观迁移
- 远程办公/数字化生活趋势
- 隐私意识和数据伦理态度

Technological（技术）
- AI/LLM 能力边界变化
- 基础设施成熟度（5G、云计算）
- 开源生态和技术标准演进
- 技术人才供需

Environmental（环境）
- 碳中和/ESG 监管要求
- 数据中心能耗限制
- 供应链可持续性要求
- 极端天气对运营的影响

Legal（法律）
- 数据保护法规（GDPR、个保法）
- 反垄断和平台监管
- 知识产权保护变化
- 行业特定许可证要求
```

### 影响矩阵

```markdown
           低量级        中量级        高量级
短期    监测即可     制定应对方案    立即行动
中期    定期扫描     纳入路线图     战略调整
长期    保持关注     情景规划       重新评估商业模式
```
