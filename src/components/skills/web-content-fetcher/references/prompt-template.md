# 抓取前 Prompt 模板（7 段）

```text
TASK:
一句话说明要解决的核心问题。

CONTEXT:
背景、受众、业务场景和已有信息。

SPECIFIC QUESTIONS:
需要回答的 3-8 个子问题。

KEYWORDS:
检索关键词与同义词。

CONSTRAINTS:
地域、时间窗口、可信来源范围、排除项。

OUTPUT:
输出格式（摘要/对比表/时间线/要点列表）和长度。

FINAL_INSTRUCTIONS:
引用规范、冲突处理、未知项显式标注方式。
```

## 最低质量要求

- 不留占位符
- 时间和地域必须可执行
- 输出格式明确到结构而非“详细一点”
