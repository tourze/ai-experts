# 活动、文化与休闲预设

## 66. 博物馆 / 美术馆
- 风格：夸张极简 + 编辑风
- 配色：`#18181B` / `#F8F5EF` / 背景 `#FFFFFF`
- 字体：Tenor Sans / Lora
- 效果：大图作品、展览时间线、票务清晰
- 避免：活泼动效、密集信息

## 67. 剧院 / 表演艺术

- 避免：现代极简（失去戏剧感）
