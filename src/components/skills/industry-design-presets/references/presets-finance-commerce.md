# Finance & Commerce Presets

- Colors: `#1E3A8A` / `#059669` growth green / bg `#F8FAFC`

- Effects: 大数字 + 细微涨跌箭头

- Style: Cyberpunk + Glassmorphism
- Colors: `#0F172A` / `#8B5CF6` / 强 `#22D3EE` / bg `#020617`
