# 服务、酒店与创意预设

## 27. 美容 / SPA / 养生
- 风格：柔和 UI 演化 + 优雅
- 配色：`#E8B4B8` / `#A8D5BA` / 金 `#D4AF37` / 背景 `#FFF5F5`
- 字体：Cormorant Garamond / Montserrat
- 效果：柔阴影、缓慢过渡、大图
- 避免：霓虹、锐利动效、暗黑默认

## 28. 餐厅
- 风格：编辑网格 + 大图
- 配色：`#18181B` / `#FBBF24` / 背景 `#FAFAF9`
- 字体：Playfair Display / Lato
- 效果：菜品大图、预订 CTA 突出
- 避免：密集文字、卡通

## 29. 酒店 / 旅行
- 风格：视差叙事 + 大图
- 配色：`#0C4A6E` / `#F59E0B` / 背景 `#FAFAFA`
- 字体：Newsreader / Inter
- 效果：滚动叙事、地图、沉浸图
- 避免：堆信息、弱视觉

## 30. 法律 / 律师事务所
- 风格：瑞士现代 2.0 + 衬线体
- 配色：`#0F172A` / `#B08D57` / 背景 `#F8FAFC`
- 字体：Libre Caslon Text / Lato
- 效果：克制、衬线、留白
- 避免：活泼、亮色、Emoji

## 31. 上门服务 / 本地商户
- 风格：扁平 + 信任信号
- 配色：`#0284C7` / `#F59E0B` / 背景 `#FFFFFF`
- 字体：Inter / Inter
- 效果：评分/证件/团队照突出
- 避免：过精致、过简化到无信息

## 32. 设计作品集 / 设计公司
- 风格：粗野主义 + 交互光标
- 配色：`#000000` / `#FAFAFA` / 偶尔亮 accent
- 字体：Syne / Inter
- 效果：大图 1:1、自定义光标、滚动揭示
- 避免：模板感 SaaS 风、温和圆角

## 33. 摄影
- 风格：夸张极简 + 全屏铺满
- 配色：`#0A0A0A` / 背景 `#FFFFFF` 或 `#0A0A0A`（双模式）
- 字体：Tenor Sans / Lora
- 效果：全屏图、边缘裁切、图网格
- 避免：装饰干扰图、多配色

## 34. 游戏（网页/产品）
- 风格：复古未来 + HUD
- 配色：`#0F172A` / `#22D3EE` / 警 `#EF4444` / 背景 `#020617`
- 字体：Unbounded / Inter
- 效果：发光、动效丰富、音效 UX
- 避免：极简白、柔和感

## 35. 音乐流媒体 / 音乐人
- 风格：Vaporwave + 渐变网格
- 配色：`#BE185D` / `#8B5CF6` / 背景 `#0F0A1F`
- 字体：Monoton / Inter
- 效果：渐变背景、节拍动画
- 避免：企业感蓝、衬线正式

## 36. 社论 / 新闻 / 杂志
- 风格：编辑网格 / 杂志风
- 配色：`#18181B` / `#EF4444` accent / 背景 `#FFFFFF`
- 字体：Playfair Display / Source Sans 3
- 效果：大标题、多栏、图文混排
- 避免：居中式 SaaS hero

## 37. 代理 / 咨询公司
- 风格：编辑风 + 夸张极简
- 配色：`#0A0A0A` / 背景 `#F8F5EF` / accent `#D45A2C`
- 字体：Fraunces / Inter
- 效果：大标题、案例网格、真实客户 logo
- 避免：模板感 SaaS、过多特性列表
