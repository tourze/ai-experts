# 健康、养生与生活方式预设

## 18. 医疗诊所 / 医院
- 风格：无障碍可及 + 伦理 + 极简
- 配色：`#0891B2` / `#10B981` / 背景 `#F0FDF4`
- 字体：Source Serif 4 / Source Sans 3
- 效果：大留白、清晰层级
- 避免：深色默认、时髦酷炫、AI 渐变

## 19. 心理健康 / 冥想

- 避免：高饱和、锐利动效、紧张节奏
