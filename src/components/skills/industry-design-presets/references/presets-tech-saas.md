# 科技 / SaaS 预设

## 1. SaaS（通用）
- 风格：扁平 + 轻玻璃拟态
- 配色：`#2563EB` / `#EA580C` / 背景 `#F8FAFC`
- 字体：Inter / Inter
- 效果：微妙 hover 200ms、浅阴影卡片
- 避免：过度动画、默认暗黑、AI 紫粉渐变

## 2. 开发者工具 / IDE / CLI
- 风格：极简 + 等宽强调
- 配色：`#0F172A` / `#22D3EE` / 背景 `#020617`（深色默认）
- 字体：Geist / Geist Mono
- 效果：等宽数字、代码块、无装饰
- 避免：儿童感、AI 渐变、图标过载

## 3. AI / 聊天机器人 / Copilot
- 风格：AI 原生 + 柔和渐变
- 配色：`#0F172A` / `#8B5CF6`（克制） / 背景 渐变 `#020617→#0F172A`
- 字体：Geist / Inter
- 效果：流式生成动效、粒子/光晕
- 避免：纯紫粉渐变一条道、AI=魔法光

## 4. 网络安全
- 风格：HUD + 瑞士现代 2.0
- 配色：`#0F172A` / `#10B981` / 警 `#EF4444` / 背景 `#020617`
- 字体：IBM Plex Sans / IBM Plex Mono
- 效果：等宽数据、网格、技术图示
- 避免：卡通、渐变、过度动画

## 5. 微 SaaS / 独立开发者
- 风格：极简直接 + 鲜艳强调
- 配色：`#6366F1` / `#059669` / 背景 `#F5F3FF`
- 字体：Plus Jakarta Sans / Inter
- 效果：演示 GIF/视频置顶、极速滚动
- 避免：企业级堆特性、长篇大论

## 6. 开发者工具可观测性 / CI
- 风格：极简 + 等宽代码
- 配色：`#020617` / `#10B981` / 错 `#EF4444` / 背景 `#0F172A`
- 字体：Geist / Geist Mono
- 效果：等宽日志、彩色代码块
- 避免：卡通、光鲜

## 7. 企业 B2B 平台
- 风格：维度分层 + 数据密集
- 配色：`#0F172A` / `#3B82F6` / 背景 `#F8FAFC`
- 字体：IBM Plex Sans / IBM Plex Sans
- 效果：层级阴影、数据密集卡片
- 避免：消费级活泼、AI 紫粉

## 8. 数据 / BI / 分析
- 风格：数据密集仪表板 + 小多图
- 配色：`#1E293B` / ColorBrewer 色盲安全板 / 背景 `#F8FAFC`
- 字体：Inter / JetBrains Mono
- 效果：小多图、精细 tooltip
- 避免：3D 图、双 Y 轴（见 `data-visualization`）
