# 复古 / 玩味系

## Y2K Aesthetic（千禧年美学）
- **关键词**: 2000年代初期、镀铬、气泡、闪亮、Aqua
- **CSS 特征**: chrome 渐变 / aqua buttons / 星星图标 / 亮面质感
- **最适合**: 时尚、音乐、Z 世代
- **不适合**: 企业、严肃场景
- **AI Prompt**: "Y2K aesthetic, early 2000s chrome, aqua bubble buttons, shiny metallic, nostalgic"

## Vaporwave（蒸汽波）
- **关键词**: 柔和紫粉、复古 80 年代、故障、大理石
- **CSS 特征**: 紫粉渐变 / 大理石贴图 / VHS 效果 / 日系汉字装饰
- **最适合**: 音乐、游戏、艺术家作品集
- **不适合**: 企业、医疗
- **AI Prompt**: "vaporwave aesthetic, pastel purple pink, retro 80s, glitch VHS, marble statue"

## Memphis Design（孟菲斯设计）
- **关键词**: 80年代几何、撞色纹样、大胆色
- **CSS 特征**: 几何装饰 / 撞色 / Z 形线 / 圆点斑马纹
- **最适合**: 创意机构、音乐、青年品牌
- **不适合**: 金融、医疗、法律
- **AI Prompt**: "memphis design, 80s geometric pattern, clashing colors, zigzag polka dots"

## Pixel Art（像素艺术）
- **关键词**: 8-bit、复古游戏、块状像素、低分辨率
- **CSS 特征**: `image-rendering: pixelated` + 低分辨率 sprite + 8-bit 调色板
- **最适合**: 独立游戏、复古工具、创意作品集
- **不适合**: 生产力工具、专业服务

## Vintage Analog / Retro Film（复古模拟/胶片）
- **关键词**: 胶片颗粒、棕褐、暖调、怀旧
- **CSS 特征**: 胶片颗粒 overlay / 暖色滤镜 / 怀旧 serif / 手写注释
- **最适合**: 摄影、音乐/黑胶品牌
- **不适合**: 现代 SaaS、科技

## Chromatic Aberration / RGB Split（色差/RGB 分离）
- **关键词**: RGB 偏移、故障、边缘分离
- **CSS 特征**: `text-shadow: 2px 0 red, -2px 0 blue` / SVG filter / 鼠标动态偏移
- **最适合**: 音乐、游戏、创意
- **不适合**: 正文、信息密集型页面

## Motion-Driven（动效驱动：便于混搭查询）
- **关键词**: 滚动动画、揭示、轻量视差
- **CSS 特征**: IntersectionObserver + transform + opacity 入场
- **最适合**: 与上述任一复古风格混搭时
- **检查清单**: ☐ 尊重 `prefers-reduced-motion` ☐ 不阻塞首屏
