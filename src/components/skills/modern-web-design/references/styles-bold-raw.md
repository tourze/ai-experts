# 粗砺 / 大声量系

## Brutalism（粗野主义）
- **关键词**: 原生、粗砺、等宽字、无圆角、实用主义
- **CSS 特征**: `border-radius: 0` / `border: 2px solid #000` / 字体对比 / 裸 HTML 感 / 单色或反差色
- **最适合**: 设计作品集、艺术项目、态度品牌
- **不适合**: 银行、医疗、消费 App
- **AI Prompt**: "brutalism web design, raw harsh typography, no rounded corners, monochrome, utilitarian aesthetic"
- **检查清单**: ☐ 无圆角 ☐ 粗边框 ☐ 等宽字或 Display 粗体 ☐ 刻意"不漂亮"

## Neubrutalism（新粗野主义）
- **关键词**: 粗阴影、纯色块、厚边框、Figma 风格
- **CSS 特征**: `border: 3px solid #000` + `box-shadow: 4px 4px 0 #000`（offset solid 阴影） + 饱和色 + 大圆角对比粗边
- **最适合**: Z 世代、初创产品、Figma 时代产品
- **不适合**: 法律、企业、可访问性敏感场景
- **AI Prompt**: "neubrutalism, bold offset shadows, thick black borders, bright solid colors, playful rebellious"
- **检查清单**: ☐ 边框 ≥ 2px solid ☐ offset 阴影（非 blur） ☐ 饱和色 ≥ 2 个

## Anti-Polish / Raw Aesthetic（反精致/原生美学）
- **关键词**: 不完美、手作感、素描、独立
- **CSS 特征**: 手写字体 / 歪斜元素 / 低保真装饰 / 反设计感
- **最适合**: 创意作品集、艺术家网站、独立创作者
- **不适合**: 企业、工具类、信任密集型
- **AI Prompt**: "anti-polish raw aesthetic, imperfect handmade sketch, indie creative, low-fi decoration"

## Gen-Z Chaos / Maximalism（Z世代混乱/极繁主义）
- **关键词**: 贴纸、撞色、层叠、活力、密集
- **CSS 特征**: 多字体 / 多色 / 贴纸元素 / 歪斜 `transform: rotate(-3deg)` / 重叠
- **最适合**: Z 世代生活方式、音乐人、青年品牌
- **不适合**: B2B、企业、严肃场景
- **AI Prompt**: "gen-z maximalism, sticker collage, clashing colors, layered chaos, vibrant dense composition"

## Kinetic Typography（动态排版）
- **关键词**: 动效文字、动画字体、滚动驱动、超大字号
- **CSS 特征**: 大字号 / scroll-triggered 动画 / text-mask / variable font 动态
- **最适合**: 首屏区域、营销网站、产品发布页
- **不适合**: 正文、信息密集型页面
- **AI Prompt**: "kinetic typography, animated oversized text, scroll-driven motion, variable font"

## Editorial Grid / Magazine（编辑栅格/杂志风）
- **关键词**: 多栏、衬线标题、引述、非对称
- **CSS 特征**: 非对称栅格 / 大标题衬线 / 多栏正文 / 图文混排
- **最适合**: 新闻、博客、杂志、长文
- **不适合**: App、仪表盘、工具
- **AI Prompt**: "editorial magazine layout, multi-column grid, serif headlines, pull quotes, asymmetric"
- **检查清单**: ☐ 多栏 ☐ Display Serif ☐ 大标题 ☐ 非对称结构
