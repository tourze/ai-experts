# 未来主义 / 技术系

## Cyberpunk UI（赛博朋克界面）
- **关键词**: 霓虹、故障、暗黑、青-品红、反乌托邦
- **CSS 特征**: bg `#020617` + neon `#22D3EE`/`#EC4899` + glitch 动画 + 扫描线 + 发光
- **最适合**: 游戏、Web3、加密货币、科技产品
- **不适合**: 银行、医疗、法律
- **AI Prompt**: "cyberpunk ui, neon cyan magenta, dark dystopian, glitch effects, scanlines, glowing text"
- **检查清单**: ☐ 深色背景 ☐ 饱和霓虹色 ☐ 发光 text-shadow ☐ 可选 glitch 动画

## HUD / Sci-Fi FUI（抬头显示/科幻未来界面）
- **关键词**: 抬头显示、未来 UI、网格、数据流
- **CSS 特征**: 等宽字 + 线框图 + 数据流动画 + 克制霓虹 + 网格
- **最适合**: 科幻游戏、航天科技、网络安全平台
- **不适合**: 消费 App、移动端
- **AI Prompt**: "HUD sci-fi FUI, futuristic heads-up display, wireframe grids, data stream animations"

## Retro-Futurism（复古未来主义）
- **关键词**: 80年代未来、镀铬、网格地平线、合成波
- **CSS 特征**: 紫粉渐变 + 网格地平线 + chrome text + 80年代色板
- **最适合**: 游戏、娱乐、音乐
- **不适合**: 金融科技、医疗
- **AI Prompt**: "retro-futurism, 80s synthwave, chrome text, grid horizon, purple-pink gradient"

## Aurora UI（极光界面）
- **关键词**: 流动渐变、柔光、氛围感
- **CSS 特征**: 多色 radial-gradient 叠加 + 动态变换 + 半透明 overlay
- **最适合**: 现代 SaaS、创意机构、AI 产品
- **不适合**: 密集仪表盘、严肃 B2B
- **AI Prompt**: "aurora ui, flowing soft light gradients, ambient atmospheric, multi-color blend"
- **检查清单**: ☐ ≥ 2 个 radial-gradient ☐ 低饱和 ☐ 不喧宾夺主

## Gradient Mesh / Aurora Evolved（渐变网格/极光进化）
- **关键词**: 复杂渐变、模糊网格、生成式
- **CSS 特征**: SVG/Canvas 生成 mesh + blur + 多点色彩
- **最适合**: 首屏背景、创意场景
- **不适合**: 正文区域、信息面板

## AI-Native UI（AI 原生界面）
- **关键词**: 生成式、流式、对话、Copilot、柔和光晕
- **CSS 特征**: 流式生成动画（逐字打字） + 低饱和渐变 + 粒子/光晕 + Mono 字体混用
- **最适合**: AI 聊天机器人、Copilot、LLM 平台
- **不适合**: 非 AI 产品（套 AI 风会显得蹭热度）
- **AI Prompt**: "AI-native ui, generative streaming interface, soft glow, copilot chat, subtle gradient"
- **务必避免**: 纯紫粉渐变一条道、把 AI 等同于魔法光

## Spatial UI（空间界面：VisionOS 风格）
- **关键词**: 3D 空间、深度、玻璃态、眼动追踪
- **CSS 特征**: Glass + 3D transform + 景深模糊 + 大圆角
- **最适合**: 空间计算、VR/AR、Apple Vision Pro
- **不适合**: 2D Web、移动端
