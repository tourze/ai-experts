## 不适用场景
- 新品牌（< 6 个月）：还没有足够的市场数据做五维评估，先聚焦产品和获客。
- 需要具体营销执行方案：品牌诊断找问题，执行方案用 [content-strategy](../content-strategy/SKILL.md)。

## 检查清单
- [ ] 五个维度都有数据或定性判断。
- [ ] 有对标（竞品或行业均值）。
- [ ] 找到了漏斗收窄点。
- [ ] 品牌类型判断准确，对策匹配。

## 反模式

### FAIL: 认知度 = 品牌力

```
"品牌认知度 80%，品牌很好"
-> 认知 80% / 美誉 30% / 购买 5% = 虚名品牌
-> 知道但不买比不知道更糟
```

### PASS: 漏斗诊断

```
认知 60% -> 美誉 70% -> 使用 40% -> 购买 15% -> 推荐 NPS 45
                                      ^^ 收窄：使用到购买转化仅 37.5%
诊断：用过但不买 -> 试用体验好但价格/付费流程有摩擦
行动：简化付费流程（7 步 -> 3 步）+ 14 天免费试用自动续费 + ROI 计算器
```
