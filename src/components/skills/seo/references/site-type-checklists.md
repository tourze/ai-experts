# 按站点类型的 SEO 检查清单

不同类型站点有不同的常见 SEO 问题和优先级。审计时先确定站点类型，再使用对应清单。

---

## SaaS / 产品站

| 检查项 | 说明 |
|--------|------|
| 产品页内容深度 | 功能页是否只有一句话描述？应包含用例、截图、数据 |
| 博客与产品页互链 | 博客文章是否链接到相关功能页？功能页是否引用博客深度内容？ |
| 对比页 / 替代页 | 是否有 "[竞品] vs [你]"和 "[竞品] alternatives" 页面？ |
| 集成页 | 每个集成是否有独立、有内容的页面（不只是 logo 墙）？ |
| 术语表 / 教育内容 | 是否有针对行业术语的教育页面，用于捕获信息意图流量？ |
| 定价页 SEO | 定价页是否可索引？是否有 Product Schema？ |
| 注册流打断 | 注册流中的页面是否被意外索引？应 noindex |

---

## 电商站

| 检查项 | 说明 |
|--------|------|
| 分类页内容 | 分类页是否只有商品列表？应包含类目描述、选购指南 |
| 商品描述重复 | 是否直接使用供应商描述？应有独特描述 |
| Product Schema | 是否有价格、库存、评分的结构化数据？ |
| 筛选导航 | 筛选组合是否产生大量重复 URL？应 canonical 或 noindex |
| 下架商品 | 下架商品是否 404？应 301 到替代品或分类页 |
| 分页 | 分页是否有 rel=next/prev 或无限滚动的 SEO 回退？ |
| 用户评价 | 是否有 UGC 评价并标记 Review Schema？ |

---

## 内容站 / 博客

| 检查项 | 说明 |
|--------|------|
| 内容过期 | 超过 12 个月未更新的文章是否标注了日期？内容是否仍准确？ |
| 关键词蚕食 | 多篇文章是否在竞争同一个关键词？应合并或差异化 |
| 主题聚类 | 是否有 pillar page + supporting articles 的聚类结构？ |
| 内链密度 | 是否有孤岛页面（无内链指向）？ |
| 作者页面 | 是否有独立作者页面展示资质（E-E-A-T 信号）？ |
| 标签/分类页 | 标签页是否有独特内容，还是只有文章列表？瘦内容应 noindex |

---

## 本地商户

| 检查项 | 说明 |
|--------|------|
| NAP 一致性 | 名称/地址/电话在全站和第三方目录是否一致？ |
| LocalBusiness Schema | 是否有包含地理坐标、营业时间、服务区域的结构化数据？ |
| Google Business Profile | GMB 是否已认领并优化？与网站信息是否一致？ |
| 地区页面 | 多门店是否有各自的地区页面（非重复内容）？ |
| 本地内容 | 是否有与本地社区、事件相关的内容？ |
| 评价管理 | 是否在 Google、Yelp 等平台积极获取评价？ |

## 技术 SEO 通用清单

适用于所有站点类型：

### 可抓取性
- robots.txt 是否有意外屏蔽？
- XML Sitemap 是否存在、已提交、只包含可索引 URL？
- 重要页面是否在首页 3 次点击内可达？

### 可索引性
- 是否有意外 noindex？
- canonical 方向是否正确？
- 是否有重定向链/环？
- HTTP → HTTPS 重定向是否完整？
- www 与非 www 是否统一？

### Core Web Vitals
- LCP < 2.5s / INP < 200ms / CLS < 0.1
- 服务器响应时间、图片优化、JS 执行、缓存策略

### 移动端
- 响应式设计（非独立 m. 站点）
- 触控目标尺寸、视口配置、无水平滚动
