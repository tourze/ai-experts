# Apple Notes CLI 命令速查

## 查看与搜索

```bash
memo notes
memo notes --folder "Work"
memo notes --search "退款"
memo notes --view 3
```

## 新增、编辑与移动

```bash
memo notes --folder "Work" --add
memo notes --folder "Work" --edit
memo notes --folder "Work" --delete
memo notes --move
```

## 文件夹与导出

```bash
memo notes --flist
memo notes --folder "Archive" --remove
memo notes --export
```
