# Rust 文档段落速查

| 段落 | 何时需要 | 内容 |
|------|----------|------|
| 摘要行 | 所有公共项 | 一句话说明功能和主要语义。 |
| `# Examples` | 非显而易见的 API | 可运行的使用示例，作为文档测试维护。 |
| `# Errors` | 返回 `Result` 时 | 列出每种 `Err` 变体或错误条件。 |
| `# Panics` | 可能 panic 时 | 说明触发 panic 的输入或前置条件。 |
| `# Safety` | `unsafe fn` | 调用方必须满足的不变量。 |

注释 `//` 解释为什么；文档 `///` 解释公共 API 做什么、怎么用、失败条件是什么。
