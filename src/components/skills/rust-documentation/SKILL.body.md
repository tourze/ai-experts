## 文档段落速查

| 段落 | 何时需要 | 内容 |
|------|----------|------|
| 摘要行 | 所有公共项 | 一句话说明功能 |
| `# Examples` | 非显而易见的 API | 可运行的使用示例 |
| `# Errors` | 返回 `Result` 时 | 列出每种 `Err` 变体的触发条件 |
| `# Panics` | 可能 panic 时 | 说明触发 panic 的输入 |
| `# Safety` | `unsafe fn` | 调用方必须满足的不变量 |

代码示例见 [chapter_08.md](references/chapter_08.md)。
