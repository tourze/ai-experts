## 适用场景

- 需要回复评论、私信或设计忠粉维护动作。
- 想从“发完就走”变成“持续互动”，提高粉丝粘性和复访率。
- 需要围绕粉丝互动反推选题、内容节奏或社群活动。
- 需要把公域粉丝进一步沉淀到私域时，参考 [references/private-domain.md](references/private-domain.md)。
- 需要统一品牌语气和人设时，参考 [references/personal-branding-advanced.md](references/personal-branding-advanced.md)。

## 核心约束

- 先解决“回复是否及时”和“回复是否有信息量”，不要只追求高频复制粘贴。
- 评论区和私信中的承诺必须真实可交付，不要为了互动率乱开口子。
- 粉丝运营的目标是提高长期信任，不是短期刷互动。
- 对外表达必须与账号定位一致；账号主张模糊时，先回到 [xiaohongshu-commercial-growth](../xiaohongshu-commercial-growth/SKILL.md) 确认定位。
- 涉及站外导流前，先检查平台安全边界，参考 [copywriting](../copywriting/SKILL.md) 中的社交平台内容安全过滤流程。

## 代码模式

高质量评论回复模板：

```text
@用户名 先回答你的问题：<直接结论>。
如果你现在的情况是 <场景>，优先做 <动作>。
你更关心 <选项 A> 还是 <选项 B>？我可以继续展开。
```

新粉欢迎语模板：

```text
嗨，我是 <账号名>。
这里主要分享 <核心内容>，你如果正在解决 <问题>，可以先看我主页置顶。
后面你想看哪类内容，也可以直接回我。
```

每周互动复盘模板：

```markdown
## 本周互动数据
- 新增评论：
- 回复率：
- 私信咨询：
- 被反复问到的话题：

## 下周动作
- 继续放大：
- 停止重复：
- 新增互动实验：
```

## 检查清单

- 评论回复先给结论，再给补充信息。
- 回复内容中至少有一个“继续对话”的钩子问题。
- 新粉欢迎语没有夸张承诺或站外导流冲动。
- 已把高频问题沉淀成后续选题或 FAQ。

## 反模式

### FAIL: 无信息回复

```
评论：「请问适合 30 岁吗？」
回复：「谢谢支持～」
→ 用户问题没解决 / 互动率虚高 / 复访不增
```

### PASS: 直接结论 + 钩子

```
评论：「请问适合 30 岁吗？」
回复：「适合，重点是看你做的是 [A] 还是 [B]。
你目前更偏哪个？我可以接着展开。」
→ 给结论 + 留对话钩子
```

### FAIL: 评论区甩链接

```
评论：「不错」
回复：「加我微信 xxx，朋友圈每天分享」
→ 平台限流 / 粉丝抗拒 / 账号风险
```

### PASS: 价值置顶

```
评论：「不错」
回复：「谢谢，下一篇会讲 [对你有用的相关主题]」
私域承接走主页置顶 / 资料引导，不在评论区直接甩
```
