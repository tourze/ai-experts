## 代码模式

高质量评论回复模板：

```text
@用户名 先回答你的问题：<直接结论>。
如果你现在的情况是 <场景>，优先做 <动作>。
你更关心 <选项 A> 还是 <选项 B>？我可以继续展开。
```

新粉欢迎语模板：

```text
嗨，我是 <账号名>。
这里主要分享 <核心内容>，你如果正在解决 <问题>，可以先看我主页置顶。
后面你想看哪类内容，也可以直接回我。
```


## 反模式

### FAIL: 无信息回复

```
评论：「请问适合 30 岁吗？」
回复：「谢谢支持～」
→ 用户问题没解决 / 互动率虚高 / 复访不增
```

### PASS: 直接结论 + 钩子

```
评论：「请问适合 30 岁吗？」
回复：「适合，重点是看你做的是 [A] 还是 [B]。
你目前更偏哪个？我可以接着展开。」
→ 给结论 + 留对话钩子
```

### FAIL: 评论区甩链接

```
评论：「不错」
回复：「加我微信 xxx，朋友圈每天分享」
→ 平台限流 / 粉丝抗拒 / 账号风险
```

### PASS: 价值置顶

```
评论：「不错」
回复：「谢谢，下一篇会讲 [对你有用的相关主题]」
私域承接走主页置顶 / 资料引导，不在评论区直接甩
```
