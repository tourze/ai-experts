## 工作流
1. **Brief 梳理**
   - 拆清 What / Who / When / Why / How / Where。
   - 记录目标、预算/资源、已有素材、不可触碰限制和待确认项。

2. **分析输入**
   - 产品：功能/玩法、调性、核心卖点、可信证据。
   - 市场/竞品：竞品定位、传播打法、渠道重点、可区隔机会。
   - 用户：分层、关注点、满意/不满、期待、心理动机和转化阻力。

3. **定位包装**
   - 输出“卖点 x 市场区隔 x 用户洞察”的定位句。
   - 拆 3-5 个信息支柱；如需 slogan，说明每个方向服务的定位。

4. **策略规划**
   - 总策略：一句话定义方案主轴。
   - 阶段规划：主题、目标、人群、核心事件、渠道组合、指标。
   - 节奏校验：每阶段承接上一阶段的用户状态变化。

5. **活动落地**
   - 活动卡：时间、目的、对象、流程、渠道/素材、指标、依赖、风险。
   - 资源清单：素材、页面、投放、内容、PR、社群、销售或合作方分工。

## 反模式

### FAIL: 渠道清单冒充方案

```
Q1：小红书种草
Q2：抖音挑战赛
Q3：KOL 联动
-> 不知道推广什么、打动谁、为什么这些渠道优先。
```

### PASS: 先定策略主轴再排期

```
定位：面向高压职场新人，主打“低成本获得即时陪伴感”的轻社交产品。
阶段 1：用真实场景内容建立共鸣。
阶段 2：用创作者挑战放大表达欲。
阶段 3：用好友邀请机制承接转化。
-> 渠道和活动都服务同一条用户心理路径。
```
