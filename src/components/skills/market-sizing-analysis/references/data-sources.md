# 市场规模数据来源

经过筛选的可信市场调研和规模分析来源清单。

## 行业研究报告

### 高端研究机构

**Gartner** (https://www.gartner.com)

- 技术市场预测和规模估算
- 竞争定位的魔力象限
- 典型价格：每份报告 $5K-$50K
- 最适合：企业软件、IT 服务、新兴技术

**Forrester** (https://www.forrester.com)

- 业务技术和数字化转型
- 供应商对比的 Wave 评估
- 典型价格：每份报告 $3K-$30K
- 最适合：营销技术、客户体验、B2B

**IDC** (https://www.idc.com)

- IT 市场情报和规模估算
- 详细的细分市场拆分
- 典型价格：每份报告 $4K-$40K
- 最适合：硬件、软件、IT 服务

**McKinsey** (https://www.mckinsey.com/featured-insights)

- 免费洞见和报告
- 战略性行业分析
- 最适合：行业趋势、宏观经济背景

### 可获取的调研资源

**Statista** (https://www.statista.com)

- 价格：个人 $39/月，企业 $199/月
- 覆盖范围：跨行业的 80,000+ 个主题
- 最适合：快速市场规模估算、图表、趋势

**CB Insights** (https://www.cbinsights.com)

- 价格：定制价格（通常 $10K+/年）
- 覆盖范围：风险投资、创业公司市场
- 最适合：新兴市场、竞争情报

**PitchBook** (https://pitchbook.com)

- 价格：机构定价
- 覆盖范围：私人公司估值、并购、VC
- 最适合：创业公司估值、融资趋势

**Grand View Research** (https://www.grandviewresearch.com)

- 价格：每份报告 $2K-$5K
- 覆盖范围：B2C 和新兴市场
- 最适合：消费市场、医疗保健、清洁技术

## 数据质量检查清单

评估来源时：

**权威性**

- [ ] 谁发布了这项研究？
- [ ] 他们的声誉如何？
- [ ] 他们有行业专业知识吗？

**方法论**

- [ ] 数据是如何收集的？
- [ ] 样本量是多少？
- [ ] 研究是什么时候进行的？

**时效性**

- [ ] 数据是否是最新的（2 年以内）？
- [ ] 市场是否发生了重大变化？
- [ ] 增长率是否仍然适用？

**一致性**

- [ ] 多个来源是否一致？
- [ ] 定义是否一致？
- [ ] 数据能否三角验证？

**相关性**

- [ ] 是否符合你的市场定义？
- [ ] 地理范围是否合适？
- [ ] 细分是否对齐？

## 免费与付费策略

**从免费来源开始：**

1. 政府数据用于客户计数
2. 上市公司文件用于细分市场收入
3. 行业协会用于行业趋势
4. Google Scholar 用于学术研究

**在以下情况升级到付费：**

- 筹集机构融资（投资者期望高端来源）
- 需要详细的细分市场拆分
- 市场属于利基或新兴领域
- 免费来源过时或不足

**经济有效的方法：**

- 购买 1-2 份覆盖核心市场的关键报告
- 使用免费来源进行三角验证
- 用一手研究补充（客户访谈）
- 混合引用免费和付费来源

## 引用最佳实践

在市场分析中始终注明来源：

**格式：**

```
市场规模：$X.XB
来源：[出版商], [报告名称], [日期]
URL：[如有链接]
```

**示例：**

```
邮件营销软件 TAM：$7.5B (2024)
来源：Gartner, "Market Share: Email Marketing Software, Worldwide, 2024"
注：包括全球所有邮件营销软件收入
```

**需包括：**

- 出版商和报告名称
- 出版日期
- 地理范围和覆盖范围
- 所做的任何调整
- 来源链接（如公开）

## 紧急研究指南

**需要在 2 小时内获得市场规模？**

1. **查看 Statista**（15 分钟）- 快速行业概览
2. **查找上市公司**（30 分钟）- 从 10-K 获取细分市场收入
3. **LinkedIn 搜索**（20 分钟）- 统计潜在 B2B 客户数量
4. **Google Scholar**（20 分钟）- 查找学术论文
5. **自下而上计算**（30 分钟）- 客户数 × 价格
6. **三角验证**（15 分钟）- 比较来源

**记录一切：**

- 写下所有来源
- 注明所有假设
- 展示你的方法论
- 对数据质量给出说明

有一个带有明确局限性的可辩护估算，总比完全没有数据要好。
