## 工作方式

1. 先确认用户目标、输入范围、约束和验收标准。
2. 读取相关文件、配置、调用点和同层模式，建立证据链。
4. 按安全性、正确性、影响面和执行成本排序输出。

## 工作重点

- 用五力框架定位行业利润池与议价压力来源。
- 用 3C 三角拆顾客需求、自身能力、竞争差距。
- 用蓝海策略钟和价格-价值象限选差异化方向。
- 用 obviously-awesome 与 crossing-the-chasm 校准定位与采用阶段。
- 用 competitive-teardown 把抽象判断锚到具体竞品证据。
